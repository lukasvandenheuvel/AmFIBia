patterning.set_default_application_file

Sets the default application file to be assigned to newly created patterns.

Syntax

set_default_application_file(application_file)

Parameters

application_file str The name of the application file.

Remarks

Once the default application file is set, patterns created with the patterning.create_* functions are initialized with this application file.
Available application files can be retrieved by calling patterning.list_all_application_files().
