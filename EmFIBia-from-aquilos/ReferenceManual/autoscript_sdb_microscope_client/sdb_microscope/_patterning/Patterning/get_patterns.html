patterning.get_patterns

Returns a list of patterns created in the active view for the current imaging device.

Syntax

get_patterns()

Return value

List[Union] The list of existing patterns.

Remarks

  • This function provides a way to access patterns that have been created outside of AutoScript, either in Microscope Control or any other application with pattern-creation capabilities.
  • The returned list is a heterogeneous collection, containing items of different types corresponding to individual patterns, for example [RectanglePattern, LinePattern].
  • The "Trapezoid" pattern type, available in certain SEM systems and platform software versions via the Microscope Control application, is internally represented as a polygon pattern. This is how it will be included in the returned collection.

Examples


        from autoscript_sdb_microscope_client.dynamic_objects import *
        
        # Read all patterns from the active view
        all_patterns = microscope.patterning.get_patterns()
        for pattern in all_patterns:
            if type(pattern) is RectanglePattern:
                # Perform rectangle pattern specific operations
                # ...
                pass
            elif type(pattern) is CirclePattern:
                # Perform circle pattern specific operations
                # ...
                pass
