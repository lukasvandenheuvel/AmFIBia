patterning.create_line

Creates a line pattern.

Syntax

create_line(start_x, start_y, end_x, end_y, depth)

Parameters

start_x float The X coordinate of the pattern start position.
start_y float The Y coordinate of the pattern start position.
end_x float The X coordinate of the pattern end position.
end_y float The Y coordinate of the pattern end position.
depth float The depth of the pattern.

Return value

LinePattern An object representing the new line pattern.
