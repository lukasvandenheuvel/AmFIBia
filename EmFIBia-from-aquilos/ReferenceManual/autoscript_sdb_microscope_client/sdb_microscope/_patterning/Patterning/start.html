patterning.start

Starts a patterning job asynchronously.

Syntax

start()

Remarks

The function returns immediately after the patterning job is started. To control the execution of a patterning job, use functions:
  • patterning.pause()
  • patterning.resume()
  • patterning.stop()
To execute the patterning job synchronously, use patterning.run().
