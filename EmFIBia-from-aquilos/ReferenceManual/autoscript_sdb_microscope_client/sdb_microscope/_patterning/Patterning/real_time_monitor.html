patterning.real_time_monitor

Enables control and provides the status of the Real Time Monitor (RTM).

Properties

mode Gets or sets the resolution mode of Real Time Monitor. Maps to RtmMode enumeration.

Methods

get_data() Retrieves grey scale values of pattern points in a pattern or multiple patterns.
get_positions() Retrieves positions of all pattern points in a pattern or multiple patterns.
restart() Restarts the real time monitoring process.
start() Starts the real time monitoring process.
stop() Stops the real time monitoring process.

Examples


        microscope.patterning.real_time_monitor.mode = RtmMode.HIGH_RESOLUTION
        microscope.patterning.real_time_monitor.start()
        
        rtm_positions = []
        
        # Configure and start patterning here
        # ...
        
        while microscope.patterning.state != PatterningState.IDLE:
            rtm_data = microscope.patterning.real_time_monitor.get_data()
            if len(rtm_positions) == 0:
                rtm_positions = microscope.patterning.real_time_monitor.get_positions()
        
            if len(rtm_positions) >  0 and len(rtm_data) > 0:
                # Process RTM positions and data
                # ...
        
        microscope.patterning.real_time_monitor.stop()
