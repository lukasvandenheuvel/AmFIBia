patterning.create_polygon

Creates a polygon pattern.

Syntax

create_polygon(vertices, depth)

Parameters

vertices List[Point | list | tuple] The list of polygon vertices as points, where each point specifies X and Y coordinates of the vertex. The minimum number of vertices is 3.
depth float The depth of the pattern.

Return value

PolygonPattern An object representing the new polygon pattern.

Examples


        # Create a triangular-shaped pattern with a depth of 1 micron.
        vertices = [[2e-6, 2e-6], [-2e-6, 2e-6], [0, -2e-6]]
        polygon = microscope.patterning.create_polygon(vertices, 1e-6)
