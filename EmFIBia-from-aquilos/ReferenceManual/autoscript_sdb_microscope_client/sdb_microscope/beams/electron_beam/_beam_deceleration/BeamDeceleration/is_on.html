beams.electron_beam.beam_deceleration.is_on

Returns True if the beam deceleration is turned on.

Data type

[Read only] bool

Remarks

  • FIB imaging/patterning cannot be done when beam deceleration is active (On).
  • EasyLift can not be inserted when beam deceleration is active (On).
  • Beam deceleration can not be enabled if EasyLift is inserted.
