beams.electron_beam.working_distance.set_value_no_degauss

Sets the value of the working distance with the automated Immersion lens degauss procedure disabled.

Syntax

set_value_no_degauss(value)

Parameters

value float A new value of the working distance in meters.

Remarks

The method can be used to accelerate the operation of setting the working distance.

Because the degauss procedure is disabled when this method is called, it is strongly adviced to set the value using property working_distance.value after the last call of the method set_value_no_degauss(). When the property is set, the degauss procedure will be called automatically.
