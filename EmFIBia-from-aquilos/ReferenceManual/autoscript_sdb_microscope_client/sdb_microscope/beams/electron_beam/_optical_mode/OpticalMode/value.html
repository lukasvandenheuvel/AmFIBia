beams.electron_beam.optical_mode.value

Gets or sets the electron beam optical mode.

Data type

str

Remarks

The enumeration OpticalMode can be used to list all optical modes.

Examples


        # Activate "Immersion" optical mode
        microscope.beams.electron_beam.optical_mode.value = OpticalMode.IMMERSION
