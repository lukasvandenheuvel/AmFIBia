beams.electron_beam.scanning.resolution.available_values

Returns the list of valid scanning resolutions.

Data type

[Read only] List[str]

Remarks

The returned list contains only "standard" resolution presets. These resolutions can be set active via the beams.electron_beam.scanning.resolution.value property and will be used later for imaging with grab_frame() or grab_frame_to_disk() functions if resolution is not explicitly specified by parameter.
Most Thermo Fisher SEM and DualBeam systems are capable of scanning also at non-standard resolutions, which are of any width and height. Non-standard resolutions are specified for a single grab-frame action by a parameter to grab_frame() or grab_frame_to_disk() functions.
