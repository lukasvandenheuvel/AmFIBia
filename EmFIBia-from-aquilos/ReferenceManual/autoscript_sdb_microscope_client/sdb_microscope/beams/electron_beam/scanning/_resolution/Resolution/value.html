beams.electron_beam.scanning.resolution.value

Gets or sets the scanning resolution for the image acquisition.

Data type

str

Remarks

This property accepts the "standard" resolution presets, which means it can only be set to values from the ScanningResolution enumeration. It's designed to configure ion beam scanning specifically for live image acquisition, which is restricted to "standard" resolutions.

However, when performing single image acquisitions with the function imaging.grab_frame(), you have the flexibility to choose arbitrary resolutions. The resolution for each grab frame action is specified individually within the GrabFrameSettings parameter.

Examples


        # Change scanning resolution of the electron beam to 1536x1024
        microscope.beams.electron_beam.scanning.resolution.value = ScanningResolution.PRESET_1536X1024
