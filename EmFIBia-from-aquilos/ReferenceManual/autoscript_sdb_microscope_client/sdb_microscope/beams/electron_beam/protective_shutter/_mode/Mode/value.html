beams.electron_beam.protective_shutter.mode.value

Gets or sets the mode of the protective shutter. Accepts values from the ProtectiveShutterMode enumeration.

Data type

str

Examples


        # Configure the shutter to insert and retract automatically based on system conditions
        microscope.beams.electron_beam.protective_shutter.mode.value = ProtectiveShutterMode.AUTOMATIC
