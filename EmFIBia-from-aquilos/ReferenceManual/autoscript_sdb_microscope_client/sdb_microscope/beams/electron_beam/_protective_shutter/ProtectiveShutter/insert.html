beams.electron_beam.protective_shutter.insert

Inserts the protective shutter.

Syntax

insert()

Remarks

  • The function is available on the following systems: Scios (platform software version 7.x, 13.x, 20.x), Aquilos (software version 13.x, 20.x, 32.x), and Helios (software version 10.x, 12.x, 17.x, 26.x). However, for the Plasma FIB version of Helios systems (XT 9.x, 14.x, 17.2x), manual insertion of the protective shutter is disallowed in AutoScript for system safety reasons.
  • If the shutter is already inserted, the function will have no effect.
