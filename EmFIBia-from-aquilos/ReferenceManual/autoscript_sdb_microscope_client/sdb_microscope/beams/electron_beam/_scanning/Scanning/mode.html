beams.electron_beam.scanning.mode

Enables control and provides the status of the scanning mode.

Properties

value [Read only] Returns the current beam scanning mode.

Methods

set_crossover() Activates the crossover scanning mode.
set_external() Activates external control of the scanning system.
set_full_frame() Activates the full frame scanning mode.
set_line() Activates the line scanning mode.
set_reduced_area() Activates the reduced area scanning mode.
set_spot() Activates the spot scanning mode.

Remarks

The object provides methods for changing the scanning mode and setting additional parameters of the scanning modes.
