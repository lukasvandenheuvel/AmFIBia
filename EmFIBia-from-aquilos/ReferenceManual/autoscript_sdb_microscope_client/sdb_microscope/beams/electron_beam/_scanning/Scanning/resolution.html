beams.electron_beam.scanning.resolution

Enables control and provides the status of the scanning resolution.

Properties

available_values [Read only] Returns the list of valid scanning resolutions.
height [Read only] Returns the Y size of the scanning resolution. The property is read-only, to set the resolution use the Value property on this object.
value Gets or sets the scanning resolution for the image acquisition.
width [Read only] Returns the X size of the scanning resolution. The property is read-only, to set the resolution use the Value property on this object.
