beams.electron_beam.crossover_zoom.value

Gets or sets the current crossover zoom. The zoom value doesn't have a physical unit.

Data type

float

Examples


        # Set zoom to 0.62
        microscope.beams.electron_beam.crossover_zoom.value = 0.62
        
        # Read out actual zoom value
        zoom = microscope.beams.electron_beam.crossover_zoom.value
        print("crossover zoom is", zoom)
        
        # Determine maximum zoom value
        zoom_max = microscope.beams.electron_beam.crossover_zoom.limits.max
        print("maximum crossover zoom is", zoom_max)
