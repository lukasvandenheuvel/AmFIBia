beams.ion_beam.scanning

Enables control provides the status of beam scanning properties.

Classes

dwell_time Enables control and provides the status of the scanning dwell time.
mode Enables control and provides the status of the scanning mode.
resolution Enables control and provides the status of the scanning resolution.
rotation Enables control and provides the status of the scan rotation.

Properties

bit_depth Controls the bit depth used during live image acquisition.
line_integration Controls line integration during live image acquisition.
scan_interlacing Controls scan interlacing during live image acquisition.
