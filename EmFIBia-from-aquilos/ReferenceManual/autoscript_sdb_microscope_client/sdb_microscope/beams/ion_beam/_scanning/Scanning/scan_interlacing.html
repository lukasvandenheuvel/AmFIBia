beams.ion_beam.scanning.scan_interlacing

Controls scan interlacing during live image acquisition.

Data type

int

Remarks

This function splits an imaging area into a large number of blocks defined by the number of lines (from 2 to 8). The first line of each block is scanned, followed by the second one, etc. This imaging method significantly reduces sample charging.

Set this value to 1 to disable the functionality.
