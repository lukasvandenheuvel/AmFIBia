beams.ion_beam.scanning.mode.set_spot

Activates the spot scanning mode.

Syntax

set_spot()
set_spot(x, y)

Parameters

x float The X coordinate of the spot. The valid range of the coordinate is [0, 1].
y float The Y coordinate of the spot. The valid range of the coordinate is [0, 1].

Remarks

Once this scanning mode is active, use methods blank() and unblank() to control the acquisition.

To set the spot position, pass position definition as the method parameters. If no parameters are passed, the last spot position is used.

Examples


        microscope.beams.ion_beam.scanning.mode.set_spot(0.6, 0.34)
        microscope.beams.ion_beam.unblank()
        # ...
        # Do desired operations here
        # ...
        microscope.beams.ion_beam.blank()
