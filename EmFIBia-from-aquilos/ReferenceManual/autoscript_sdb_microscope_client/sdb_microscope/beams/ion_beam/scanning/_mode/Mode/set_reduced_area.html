beams.ion_beam.scanning.mode.set_reduced_area

Activates the reduced area scanning mode.

Syntax

set_reduced_area()
set_reduced_area(left, top, width, height)

Parameters

left float Left side of the rectangular area. The coordinate is in the range [0, 1].
top float Top side of the rectangular area. The coordinate is in the range [0, 1].
width float Width of the rectangular area. The valid range is [0, 1], where 1 means the full width of the view.
height float Height of the rectangular area. The valid range is [0, 1], where 1 means the full height of the view.

Remarks

The parameterless function activates the reduced area scan mode at the previously set position or the default position if called for the first time.
If rectangle coordinates are passed, the function sets the new position and then activates the mode.

To specify the reduced area position for a single image acquisition, provide the coordinates to the function grab_frame() through the settings parameter.

Examples


        # Activate reduced area scan mode while simultaneously setting the area position.
        # The reduced area will be centered in the field of view and occupy half of its size.
        microscope.beams.ion_beam.scanning.mode.set_reduced_area(0.25, 0.25, 0.5, 0.5)
        microscope.imaging.start_acquisition()
