beams.electron_beam.protective_shutter

Functions related to the retractable shutter that protects the SEM column during FIB milling.

Classes

mode Allows you to control the mode of the automatic protective shutter.

Properties

is_installed [Read only] Returns True if the system is equipped with the protective shutter.
state [Read only] Returns the current state of the protective shutter, providing values from the RetractableDeviceState enumeration.

Methods

insert() Inserts the protective shutter.
retract() Retracts the protective shutter.

Remarks

  • Manual control of the protective shutter is not supported on Helios PFIB systems.
  • To insert the protective shutter, make sure that the automatic mode is turned off.
  • It is important to ensure that the stage is not positioned too close to the electron column, as this may prevent the insertion of the shutter.

Examples


        # Turn off the automatic mode
        microscope.beams.electron_beam.protective_shutter.mode.value = ProtectiveShutterMode.OFF
        
        # Insert the protective shutter
        microscope.beams.electron_beam.protective_shutter.insert()
