beams.electron_beam.beam_mode

Enables control and provides the status of the electron beam mode.

Properties

available_values [Read only] Returns the list of modes of the electron beam source. The property returns only modes applicable at the given time. The enumeration ElectronBeamMode lists all existing modes.
value Gets or sets the electron beam mode. This property controls the "UC On" function for some of the electron beam columns.

Examples


        # Switch electron beam source to UC mode. Note that HV and beam current must be in a range that allows this mode.
        microscope.beams.electron_beam.beam_mode.value = ElectronBeamMode.UC
        print(microscope.beams.electron_beam.beam_mode.value)
        
        # Switch off UC mode.
        microscope.beams.electron_beam.beam_mode.value = ElectronBeamMode.NORMAL
        print(microscope.beams.electron_beam.beam_mode.value)
