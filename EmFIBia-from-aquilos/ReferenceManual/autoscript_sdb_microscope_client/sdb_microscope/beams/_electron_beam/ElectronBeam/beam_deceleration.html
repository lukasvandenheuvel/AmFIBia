beams.electron_beam.beam_deceleration

Enables control and provides the status of the electron beam deceleration.

Classes

stage_bias Enables control and provides the status of stage bias.

Properties

is_installed [Read only] Returns True if the beam deceleration is available on the system.
is_on [Read only] Returns True if the beam deceleration is turned on.

Methods

turn_off() Turns the beam deceleration off.
turn_on() Turns the beam deceleration on.

Remarks

Beam Deceleration applies a negative potential (stage bias) to the stage, which influences both primary and secondary electrons.
