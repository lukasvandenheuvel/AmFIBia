analysis.eds.get_peak_id

Identifies the presence of chemical elements in the given EDS spectrum.

Syntax

get_peak_id(spectrum)
get_peak_id(spectrum, excluded_elements)

Parameters

spectrum AdornedSpectrum The spectrum object that holds both the characteristic X-ray counts and system metadata.
excluded_elements List[ChemicalElement] The list of chemical elements to be excluded from peak identification.

Return value

List[ChemicalElement] The list of chemical elements found in the given EDS spectrum.

Remarks

  • In offline mode, the function does not execute actual processing of the provided spectrum. Instead, it returns simulated results - four fixed elements [Boron, Fluorine, Mercury, Thallium].
  • Some chemical elements are not supported by the AutoScript EDS quantification engine. These elements cannot be marked as excluded via the excluded_elements parameter: Helium, Neon, Krypton, Xenon, Radon, Hydrogen, Lithium, Polonium, Astatine, Actinium, Protactinium, Curium, Berkelium, Californium, Einsteinium, Fermium, Mendelevium, Nobelium, Lawrencium, Francium, Radium, Rutherfordium, Dubnium, Seaborgium, Bohrium, Hassium, Meitnerium, Darmstadtium, Roentgenium, Copernicium, Nihonium, Flerovium, Moscovium, Livermorium, Tennessine, Oganesson.

Examples


        # Acquire spectrum from the entire field of view
        region = Region(type=RegionType.FULL_FRAME)
        spectrum = microscope.analysis.eds.acquire_spectrum(region, 3)
        
        # Identify chemical elements present in the spectrum
        elements_found = microscope.analysis.eds.get_peak_id(spectrum)
        
        # Returned list: [Boron, Nitrogen, Fluorine, Gallium, Silicon]
    

        # Identify chemical elements but exclude Nitrogen from identification
        elements_found = microscope.analysis.eds.get_peak_id(spectrum, [PeriodicTable.NITROGEN])        
        
        # Returned list: [Boron, Fluorine, Gallium, Silicon]
