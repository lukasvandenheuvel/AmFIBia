analysis.eds.get_spectrum

Returns the EDS spectrum collected since the last reset of the EDS spectrum buffer.

Syntax

get_spectrum()
get_spectrum(scanning_mode)

Parameters

scanning_mode int Electron beam scan mode under which the EDS data were collected. If not provided, the current scan mode is assumed.

Return value

AdornedSpectrum The spectrum object that holds both the characteristic X-ray counts and system metadata.

Remarks

  • The returned spectrum has been collected since the last reset of the EDS spectrum buffer.
    Resetting the buffer may have been explicitly triggered by calling analysis.eds.reset_spectrum(). Changing the scanned area, e.g. the position of reduced area or spot via scanning.mode.set_reduced_area() or scanning.mode.set_spot(), also automatically resets the buffer.
  • The system maintains a separate EDS data buffer for each beam scan mode (full-frame, line, spot, reduced-area, and patterning). You don't need to specify the scanning_mode parameter if the mode hasn't changed since the data was collected.
    When a patterning sequence is executed to generate an EDS signal, the scan mode automatically reverts to full-frame after the sequence completes. To access the correct buffer in this case, you must specify the ScanningMode.PATTERNING parameter. An example below demonstrates this scenario.

Examples


        import time
        
        # Reset EDS buffers
        microscope.analysis.eds.reset_spectrum()
        
        # Scan the sample with electron beam in any of the supported scan modes,
        # e.g., in the reduced-area mode
        microscope.beams.electron_beam.scanning.mode.set_reduced_area(0, 0, 0.2, 0.2)
        microscope.imaging.start_acquisition()
        time.sleep(5)
        microscope.imaging.stop_acquisition()
    
        # Read EDS spectrum collected from the scanned area
        spectrum = microscope.analysis.eds.get_spectrum()
    

        # Create patterns in the active view 
        # ...
        
        # Reset EDS buffers
        microscope.analysis.eds.reset_spectrum()
        
        # Run the patterning job synchronously
        microscope.patterning.run()

        # Read EDS spectrum from the scanned area
        spectrum = microscope.analysis.eds.get_spectrum(ScanningMode.PATTERNING)
