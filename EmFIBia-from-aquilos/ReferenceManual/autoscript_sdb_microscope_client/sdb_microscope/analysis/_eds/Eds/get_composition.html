analysis.eds.get_composition

Performs quantification of chemical elements in the specified EDS spectrum.

Syntax

get_composition(spectrum, selected_elements)

Parameters

spectrum AdornedSpectrum The spectrum object that holds both the characteristic X-ray counts and system metadata.
selected_elements List[ChemicalElement] The list of chemical elements to be quantified.

Return value

List[ChemicalCompositionEntry] The chemical composition derived from the EDS spectrum in the form of a list of entries. Each entry gives the atomic %, weight % for a chemical element, along with additional information on the accuracy of the quantification.

Remarks

  • The elements to be quantified are usually first identified by a call to analysis.eds.get_peak_id().
  • In offline mode, the function does not perform actual processing of the provided spectrum. The composition is returned for the selected elements, but with constant simulated results.
  • Some chemical elements are not supported by the AutoScript EDS quantification engine. These elements cannot be marked as selected via the selected_elements parameter: Helium, Neon, Krypton, Xenon, Radon, Hydrogen, Lithium, Polonium, Astatine, Actinium, Protactinium, Curium, Berkelium, Californium, Einsteinium, Fermium, Mendelevium, Nobelium, Lawrencium, Francium, Radium, Rutherfordium, Dubnium, Seaborgium, Bohrium, Hassium, Meitnerium, Darmstadtium, Roentgenium, Copernicium, Nihonium, Flerovium, Moscovium, Livermorium, Tennessine, Oganesson.

Examples


        # Acquire spectrum from the entire field of view
        region = Region(type=RegionType.FULL_FRAME)
        spectrum = microscope.analysis.eds.acquire_spectrum(region, 3)
        
        # Identify chemical elements present in the spectrum
        elements_found = microscope.analysis.eds.get_peak_id(spectrum)
        
        # Get quantitative chemical composition from the spectrum
        composition = microscope.analysis.eds.get_composition(spectrum, elements_found)
        
        # Result:
        # [ChemicalCompositionEntry(element=Boron, atomic_percentage=15.56, weight_percentage=4.442),
        #  ChemicalCompositionEntry(element=Fluorine, atomic_percentage=50.11, weight_percentage=25.14),
        #  ChemicalCompositionEntry(element=Silicon, atomic_percentage=0.5068, weight_percentage=0.3759),
        #  ChemicalCompositionEntry(element=Gallium, atomic_percentage=31.64, weight_percentage=58.25)]
