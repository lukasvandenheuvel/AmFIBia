analysis.eds.mapping

Functions related to the generation of EDS elemental and phase maps.

Methods

get_stream() Returns an object that represents the currently active stream of data from the EDS detector. The data includes both the count of characteristic X-rays and positions on the sample from where the X-rays were generated.
reset_stream() Creates a new empty EDS stream object in the microscope platform software and sets it as active. EDS data generated by the subsequent beam activity, such as the number of characteristic X-rays and their positions, are collected in the newly created stream.
