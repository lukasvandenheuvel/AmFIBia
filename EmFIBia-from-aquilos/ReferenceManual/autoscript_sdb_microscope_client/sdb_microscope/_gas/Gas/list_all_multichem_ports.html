gas.list_all_multichem_ports

Returns a list of MultiChem devices installed on the system. Each device is described by a port identifier.

Syntax

list_all_multichem_ports()

Return value

List[str] The list of available MultiChem devices as the list of port identifiers.

Examples


        # Assuming the stage is linked and in a safe position
    
        # Insert all MultiChem needles
        for multichem_port in microscope.gas.list_all_multichem_ports():
            multichem = microscope.gas.get_multichem(multichem_port)
            multichem.insert()
            print(F"MultiChem at port {multichem.port_name} state: {multichem.state}")
