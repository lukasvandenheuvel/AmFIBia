detector.camera_settings.emission

Allows you to access the emission parameter of the active camera detector.

Classes

type Allows you to access the emission type parameter of the active camera detector.

Properties

is_on [Read only] Returns True if any emission is on.

Methods

start() Starts illuminating the sample with light of the wavelength defined by the emission type parameter.
stop() Stops any ongoing emission. If the emission is already turned off, this operation has no effect.
