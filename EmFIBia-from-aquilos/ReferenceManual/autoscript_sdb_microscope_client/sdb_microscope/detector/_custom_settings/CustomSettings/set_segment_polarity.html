detector.custom_settings.set_segment_polarity

Sets the polarity of a detector segment.

Syntax

set_segment_polarity(name, polarity)

Parameters

name str The name of the segment. DetectorSegment enumeration can be used to access segment names.
polarity int The new polarity of the segment. SegmentPolarity enumeration can be used to obtain polarity names.
