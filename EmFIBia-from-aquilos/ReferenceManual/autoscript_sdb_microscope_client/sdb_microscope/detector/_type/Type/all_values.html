detector.type.all_values

Retrieves all detectors types installed on the microscope.

Data type

[Read only] List[str]

Remarks

Returns a list of all installed detectors, regardless of their compatibility with the active imaging device.
Detectors not supported by AutoScript may still be omitted from the list.

Examples


        # Retract all inserted detectors
        installed_detectors = microscope.detector.type.all_values
        for detector_type in installed_detectors:
            detector_info = microscope.detector.get_info(detector_type)
            if detector_info.is_retractable and detector_info.can_retract:
                print("Retracting", detector_type)
                microscope.detector.retract()
