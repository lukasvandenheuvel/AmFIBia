detector.type.available_values

Retrieves all detector types compatible with the active imaging device.

Data type

[Read only] List[str]

Remarks

  • The returned list includes detectors that are physically present on the microscope and compatible with the active imaging device. Some may not be selectable at certain times due to the state of the microscope. AutoScript provides a way to determine if a detector can be selected (i.e., if it is enabled) using the detector.get_info() function.
  • To get a list of all installed detectors, regardless of their compatibility with the active imaging device, call detector.type.all_values.
