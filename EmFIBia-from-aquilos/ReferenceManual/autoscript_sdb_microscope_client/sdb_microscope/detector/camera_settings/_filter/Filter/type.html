detector.camera_settings.filter.type

Allows you to control the optical filter that is set on the active camera detector.

Properties

available_values [Read only] Provides the list of available optical filters.
value Reads or sets the type of optical filter to be used for camera acquisitions. Optical filters are designed to transmit specific wavelengths of light while blocking others, enabling selective imaging of fluorescence signals. Maps to CameraFilterType enumeration.
