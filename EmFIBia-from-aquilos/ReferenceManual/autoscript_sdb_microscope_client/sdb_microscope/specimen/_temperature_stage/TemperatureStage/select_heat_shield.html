specimen.temperature_stage.select_heat_shield

Selects the heat shield for the currently connected temperature stage.

Syntax

select_heat_shield(type)

Parameters

type str The heat shield type. You can use the TemperatureStageHeatShieldType enumeration.

Remarks

  • The heat shield needs to be selected before using the high vacuum heating stage.
  • This method is only available for the high vacuum heating stage.

Examples


        microscope.specimen.temperature_stage.select_heat_shield(TemperatureStageHeatShieldType.EBSD)
