specimen.temperature_stage.turn_on

Turns the temperature stage on.

Syntax

turn_on()

Remarks

  • The behavior of the function differs for older and newer SEM systems:
    • On older systems with a microscope software version below 10.0 (Quanta, Quanta FEG and Versa 3D):
      Turning on the stage has no immediate effect on the stage temperature. The stage does not start heating until the target temperature value is set to specimen.temperature_stage.temperature.target_value or the function specimen.temperature_stage.ramp() is called.
    • On newer systems with a microscope software version greater than or equal to 10.0 (Quattro, Apreo, Helios etc.):
      When you turn on the stage, it immediately enters temperature control mode - i.e. it begins to regulate the temperature to the target value previously set in the system.
  • If the stage is already on, the function has no effect.

Examples


        # Turn on the heating stage and start heating to 400K.
        # Example for an older system, e.g. Quanta with software version 6.x.
        
        microscope.specimen.temperature_stage.turn_on()
        microscope.specimen.temperature_stage.temperature.target_value = 400
    

        # Turn on the heating stage and start heating to 400K.
        # Example for a newer system, e.g. Quattro with software version 15.x.

        microscope.specimen.temperature_stage.temperature.target_value = 400
        microscope.specimen.temperature_stage.turn_on()
