specimen.temperature_stage.humidity

Enables control of humidity of wet samples.

Properties

is_on [Read only] Returns the on/off state of the humidity control.
target_value Retrieves or sets the target humidity. The value is expected to be in the range of [0, 1] corresponding to 0% - 100%.
value [Read only] Retrieves the actual humidity. The value is expected to be in the range of [0, 1] corresponding to 0% - 100%.

Methods

turn_off() Turns the humidity control off.
turn_on() Turns the humidity control on. Requires Low Vacuum or ESEM vacuum conditions.
