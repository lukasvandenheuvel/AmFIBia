specimen.temperature_stage.type

Returns the type of the currently connected temperature stage. Maps to TemperatureStageType enumeration.

Data type

[Read only] str

Remarks

  • If no temperature stage is connected, the function returns None.
  • The returned string does not specify a subtype of the stage, e.g. for heating stage the string "HeatingStage" is returned without specifying which of the 1000/1000S/1400 variants is present.

Examples


        print(microscope.specimen.temperature_stage.type)
