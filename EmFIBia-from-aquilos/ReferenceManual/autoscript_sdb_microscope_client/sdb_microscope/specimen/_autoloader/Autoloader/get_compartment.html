specimen.autoloader.get_compartment

Returns an object representing the specified cryo compartment of the Autoloader device.

Syntax

get_compartment(compartment_type)

Parameters

compartment_type str The Autoloader device compartment type. You can use values from the AutoloaderCompartmentType enumeration.

Return value

CryoCompartment An object representing the specified cryo compartment.

Examples


    # Get the Autoloader compartment
    autoloader_comparment = microscope.specimen.autoloader.get_compartment(AutoloaderCompartmentType.AUTOLOADER)
    
    # Get the chamber compartment
    chamber_compartment = microscope.specimen.autoloader.get_compartment(AutoloaderCompartmentType.CHAMBER)
