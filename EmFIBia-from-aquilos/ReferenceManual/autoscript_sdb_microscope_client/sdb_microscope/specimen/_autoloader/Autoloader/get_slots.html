specimen.autoloader.get_slots

Returns a list of slots that are available on the cassette.

Syntax

get_slots(run_inventory)

Parameters

run_inventory bool If set to True, AutoScript will perform an inventory on the cassette.

Return value

List[AutoloaderSlot] The list of slots on the Autoloader cassette.

Remarks

The inventory procedure systematically checks each slot to determine whether it is empty or occupied. One execution can take a couple of minutes.

Retrieving slots without running the inventory procedure can be useful when the inventory procedure has already been completed in the main microscope UI. This allows you to obtain the slot information without performing the inventory again, resulting in time savings.

Examples


    # Retrieve slots without running the inventory procedure
    slots = microscope.specimen.autoloader.get_slots(False)

    # Run the inventory procedure and retrieve slots
    slots = microscope.specimen.autoloader.get_slots(True)
