specimen.compustage.safety_settings.unlock_axis

Unlocks the specified Compustage axis.

Syntax

unlock_axis(axis)

Parameters

axis str, List[str] The axis or multiple axes to be unlocked.

Remarks

Axis names are not case sensitive. A valid name is e.g. "X", "x".

Examples


        # Lock axes X and Alpha, then unlock them again
        microscope.specimen.compustage.safety_settings.lock_axis('X')
        microscope.specimen.compustage.safety_settings.lock_axis('A')
        microscope.specimen.compustage.safety_settings.unlock_axis(['X', 'A'])
