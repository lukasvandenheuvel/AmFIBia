specimen.temperature_stage.humidity.turn_on

Turns the humidity control on. Requires Low Vacuum or ESEM vacuum conditions.

Syntax

turn_on()

Remarks

On Quanta and Versa 3D systems, before turning on humidity control, the temperature stage must be turned on by calling specimen.temperature_stage.turn_on().

Examples


        # Activate the humidity control mode
        microscope.specimen.temperature_stage.humidity.turn_on()
        print(microscope.specimen.temperature_stage.humidity.is_on)

        # Set target humidity to 10%
        microscope.specimen.temperature_stage.humidity.target_value = 0.1
