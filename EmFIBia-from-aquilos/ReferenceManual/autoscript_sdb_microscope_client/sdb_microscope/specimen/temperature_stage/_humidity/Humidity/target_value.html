specimen.temperature_stage.humidity.target_value

Retrieves or sets the target humidity. The value is expected to be in the range of [0, 1] corresponding to 0% - 100%.

Data type

float

Examples


        # Set target humidity to 0.5 (50%)
        microscope.specimen.temperature_stage.humidity.target_value = 0.5
        print(microscope.specimen.temperature_stage.humidity.target_value)
