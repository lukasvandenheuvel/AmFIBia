specimen.temperature_stage.temperature.target_value

Gets or sets the target temperature of the stage. The default unit is Kelvin, but you can configure a preferred unit by calling set_preferred_unit().

Data type

float

Remarks

When a new target value is set while the temperature stage is turned on, it immediately begins ramping the actual temperature up or down to the target value.
