specimen.stage.holder.region.activate

Activates the specified region and moves the stage to the designated position within that region. This function can also be utilized to move within an already activated region.

Syntax

activate(region_name, region_position)

Parameters

region_name str The name of the holder region.
region_position str The name of the position in the region to move to.

Examples

Activate a region: 

        # Activate the "Pre-Tilted Rowbar" region and move the stage to the region's default position
        microscope.specimen.stage.holder.region.activate("Pre-Tilted Rowbar", "Default")
Move stage within the active region to a different position: 

        # Activate the "Pre-Tilted Rowbar" region
        microscope.specimen.stage.holder.region.activate("Pre-Tilted Rowbar", "Default")       
        ...
        # Move stage to "Plan View Thinning w/STEM end-pointing" position
        microscope.specimen.stage.holder.region.activate("Pre-Tilted Rowbar", "Plan View Thinning w/STEM end-pointing")
