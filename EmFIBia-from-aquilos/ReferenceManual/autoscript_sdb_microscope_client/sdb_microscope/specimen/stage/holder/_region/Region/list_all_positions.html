specimen.stage.holder.region.list_all_positions

Returns a list of position names for the specified region. If no region name is provided, the function will return a list of position names for the currently active region.

Syntax

list_all_positions()
list_all_positions(region_name)

Parameters

region_name str The name of the holder region.

Return value

List[str] List of position names.

Examples


    # Get available positions for the "Fixed Bulk" region
    positions = microscope.specimen.stage.holder.region.list_all_positions("Fixed Bulk")
