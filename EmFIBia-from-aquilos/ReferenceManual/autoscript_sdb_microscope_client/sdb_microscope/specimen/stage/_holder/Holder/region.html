specimen.stage.holder.region

Provides access to holder regions and their positions.

Properties

available_values [Read only] Returns the list of available regions on the currently selected specimen holder.
is_available [Read only] Returns True if holder regions are supported on this microscope.
value [Read only] Returns the name of the currently active holder region.

Methods

activate() Activates the specified region and moves the stage to the designated position within that region. This function can also be utilized to move within an already activated region.
list_all_positions() Returns a list of position names for the specified region. If no region name is provided, the function will return a list of position names for the currently active region.
