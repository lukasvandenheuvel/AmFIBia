specimen.sample_cleaning.run

Executes the sample cleaning procedure synchronously, i.e. waits for the sample cleaning to end.

Syntax

run(time)

Parameters

time int The cleaning time in seconds.

Remarks

The specimen chamber must be pumped to high vacuum conditions before initiating the sample cleaning process.

Examples


        # Execute the sample cleaning procedure for a duration of 30 seconds if the device is installed
        if microscope.specimen.sample_cleaning.is_installed:
            microscope.specimen.sample_cleaning.run(30)
