specimen.manipulator.set_default_coordinate_system

Establishes the default coordinate system for the manipulator. If no system is explicitly specified by the position, any subsequent calls to absolute_move(), relative_move() and insert() will utilize this default system.

Syntax

set_default_coordinate_system(coordinate_system)

Parameters

coordinate_system str The new default manipulator coordinate system. You can provide values from the ManipulatorCoordinateSystem enumeration.

Remarks

Stage is the default coordinate system before this function is called.

Examples


        microscope.specimen.manipulator.set_default_coordinate_system(ManipulatorCoordinateSystem.RAW)
    
        # Move to a position using the Raw coordinate system
        position = ManipulatorPosition(x=0.0001, y=0.0001)
        microscope.specimen.manipulator.absolute_move(position)
