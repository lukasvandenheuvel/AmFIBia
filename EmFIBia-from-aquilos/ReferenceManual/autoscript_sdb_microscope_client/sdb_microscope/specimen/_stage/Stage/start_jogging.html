specimen.stage.start_jogging

Initiates the continuous movement ("jogging") of the stage, determined by the input velocity in m/s for each axis. The stage jogs until stopped or until the time interval is reached. The function is asynchronous and returns immediately.

Syntax

start_jogging(velocity)
start_jogging(velocity, duration)

Parameters

velocity StageVelocity The velocity of the stage movement.
duration float Duration in seconds specifying the jogging time for the stage.

Remarks

  • The maximum jogging speed varies for each microscope system. The tested maximum speed is 1e-3 m/s, consistent across all supported systems.
  • The direction along the axis is determined by the positive or negative velocity value.
  • If the duration parameter is omitted, the stage can be stopped by invoking specimen.stage.stop(). If not stopped explicitly, it will halt upon reaching the axis maximum.
  • Jogging functionality is not supported on systems with a platform software version older than 12.0. Furthermore, the PlutoWDB system also does not implement this function due to variations in stage hardware.

Examples


        import time
        
        # Initiate jogging at a speed of 1 mm/s along the X axis and -1 mm/s along the Y axis
        microscope.specimen.stage.start_jogging(velocity=StageVelocity(1e-3, -1e-3))

        # Do whatever is needed while the stage is jogging
        time.sleep(3)

        # Stop the stage
        microscope.specimen.stage.stop()
    

        # Initiate movement of the stage at a speed of -1 mm/s along the X axis for a duration of 10 seconds,
        # while capturing an image every 2 seconds. It is assumed that live image acquisition is in progress.
        images = []
        microscope.specimen.stage.start_jogging(StageVelocity(-1e-3), 10)
        
        for i in range(0, 5):
            time.sleep(2)
            print("Taking image...")
            images.append(microscope.imaging.get_image())
