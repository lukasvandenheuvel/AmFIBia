specimen.stage.absolute_move

Moves the stage to the specified position.

Syntax

absolute_move(target_position)
absolute_move(target_position, settings)

Parameters

target_position StagePosition The absolute position to which the stage should move.
settings MoveSettings Additional move settings.

Remarks

  • The function uses coordinate system specified by function set_default_coordinate_system(), with "Specimen" being the default coordinate system.
  • Navigating with the stage is mutually exclusive with using the compustage device (if installed). Only one of these two sample navigation devices can be active at a time. If the compustage is active, any movement with the bulk stage will deactivate the compustage and activate the stage. And the other way around.
  • The behavior of the function is independent of the positioning settings configured in the main user interface of the microscope, such as the state of the "Compucentric rotation" or "Link Z-Y" check boxes on the Navigation page.
  • If some of the move settings are not explicitly specified in settings, they will default to the values configured by a previous call to specimen.stage.set_default_move_settings() or to False if the function was not called to set the default settings.

Examples


        # Move the stage to coordinate [X, Y] = [2mm, 4mm]. Do not move with the rest of the axes.
        position = StagePosition(x=0.002, y=0.004)
        microscope.specimen.stage.absolute_move(position)
    

        # Rotate the stage compucentrically by 90 degrees
        import math
    
        position = StagePosition(r=math.radians(90))
        microscope.specimen.stage.absolute_move(position, MoveSettings(rotate_compucentric=True))
