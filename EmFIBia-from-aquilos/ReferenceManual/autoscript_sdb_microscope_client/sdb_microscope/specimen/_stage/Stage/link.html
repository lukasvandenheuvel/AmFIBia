specimen.stage.link

Links the stage Z to the free working distance.

Syntax

link()

Remarks

The function sets the Z stage coordinate value to the actual value of the free working distance (FWD) and changes the orientation of the Z axis so that it points downward. This allows accurate and safe movement between the top surface height of the specimen and the end of the objective lens.
When stage Z is linked to FWD, moving the stage in Z automatically adjusts the working distance value to keep the specimen surface in focus.
