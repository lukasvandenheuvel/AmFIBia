specimen.stage.set_default_move_settings

Sets the default settings for stage moves. Subsequent absolute_move() and relative_move() calls where MoveSettings are not specified use these settings.

Syntax

set_default_move_settings(settings)

Parameters

settings MoveSettings Move settings to be saved as new defaults.

Remarks

  • The function configures the behavior of specimen.stage.absolute_move() and specimen.stage.relative_move() calls in case the settings parameter was not explicitly specified or some of the MoveSettings items were not initialized in the passed parameter.
  • The new default values are saved only for the currently executed script. Therefore, setting the defaults does not affect the behavior of other scripts.
  • The new default values are saved only for the calling client. Thus, setting the defaults does not affect the behavior of other running API clients.
  • Before this function is called, all default move settings are initialized with False.

Examples


        # Configure the default stage move strategy to rotate compucentrically
        microscope.specimen.stage.set_default_move_settings(MoveSettings(rotate_compucentric=True))
        
        # Rotate compucentrically
        microscope.specimen.stage.absolute_move(StagePosition(r=0.2))
        
        # Rotate compucentrically
        microscope.specimen.stage.absolute_move(StagePosition(r=0.2), MoveSettings())
        
        # Rotate compucentrically
        microscope.specimen.stage.absolute_move(StagePosition(r=0.2), MoveSettings(rotate_compucentric=None))

        # Rotate compucentrically
        microscope.specimen.stage.absolute_move(StagePosition(r=0.2), MoveSettings(tilt_compucentric=True))
       
        # Rotate normally
        microscope.specimen.stage.absolute_move(StagePosition(r=0.2), MoveSettings(rotate_compucentric=False))
