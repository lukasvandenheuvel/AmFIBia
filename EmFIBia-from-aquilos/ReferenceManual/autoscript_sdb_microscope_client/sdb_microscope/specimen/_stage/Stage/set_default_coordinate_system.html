specimen.stage.set_default_coordinate_system

Sets the default coordinate system of the stage.

Syntax

set_default_coordinate_system(coordinate_system)

Parameters

coordinate_system str The stage coordinate system to set as default. You can select a coordinate system from the CoordinateSystem enumeration.

Remarks

  • The default coordinate system before the method is called is Specimen.
  • The functions specimen.stage.absolute_move() and specimen.stage.relative_move() use this system if none is specified by the position.
