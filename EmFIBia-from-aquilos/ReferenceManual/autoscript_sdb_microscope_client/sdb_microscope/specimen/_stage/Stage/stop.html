specimen.stage.stop

Immediately stops movement of the stage in all axes.

Syntax

stop()

Remarks

  • Except for the function specimen.stage.start_jogging(), all API functions for stage movement execute the moves synchronously. There is no need to manually stop a movement initiated by synchronous functions, as they return only when the stage is already stationary.
    However, external triggers, such as user interventions in the main user interface or automatic actions of the platform software, may initiate stage movement that might need to be stopped. An example of such an action is the "Z-Cleaning" procedure, which automatically moves the stage up and down for several minutes after loading the LoadLock device.
  • If the stage is not moving, the function does nothing.
