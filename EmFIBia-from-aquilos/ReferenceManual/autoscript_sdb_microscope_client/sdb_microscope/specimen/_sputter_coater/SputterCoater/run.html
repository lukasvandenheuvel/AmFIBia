specimen.sputter_coater.run

Executes the sputtering procedure.

Syntax

run(time)
run(time, grids)

Parameters

time int The sputtering time in seconds.
grids List[int] Specifies indices of the grids on which to execute the sputtering process.

Remarks

On the Arctis system, this function additionally performs the home procedure of the fluorescence light microscope (FLM) and moves the compustage to the sputtering position. After the sputtering is completed, it returns the compustage to its original position.

The grids parameter is required for the sputtering procedure on Hydra Bio systems. For other systems, it should be omitted.

Examples

Run sputtering on Aquilos system: 

        # Prepare for sputtering
        microscope.specimen.sputter_coater.prepare()

        # Change chamber pressure to 20 Pa and sputter current to 10 mA
        microscope.vacuum.pump(VacuumSettings(pressure=20))
        microscope.specimen.sputter_coater.current.value = 0.01

        # Perform the sputtering procedure with a run time of 10 seconds
        microscope.specimen.sputter_coater.run(10)

        # Recover from sputtering
        microscope.specimen.sputter_coater.recover()
Run sputtering on Hydra Bio system: 

        # Find out whether the MicroSputter is installed
        print(microscope.specimen.sputter_coater.is_installed)
        
        # Inspect the sputter current and its possible values
        print(microscope.specimen.sputter_coater.current.value)
        print(microscope.specimen.sputter_coater.current.available_values)
        
        # Prepare for sputtering
        microscope.beams.ion_beam.turn_on()
        if not microscope.specimen.sputter_coater.is_homed:
            microscope.specimen.sputter_coater.home()
        
        # Perform sputtering with a run time of 10 s at grid 1 and grid 2 positions
        microscope.specimen.sputter_coater.run(10, [1, 2])
