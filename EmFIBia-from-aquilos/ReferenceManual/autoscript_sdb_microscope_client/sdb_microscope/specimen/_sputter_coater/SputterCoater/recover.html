specimen.sputter_coater.recover

Returns the system to the state before the prepare() function was called.

Syntax

recover()

Remarks

  • The function restores the chamber vacuum mode, pressure, type of gas and the state of electron and ion beams to their original values.
  • The function is not supported on Arctis or Hydra Bio systems.
