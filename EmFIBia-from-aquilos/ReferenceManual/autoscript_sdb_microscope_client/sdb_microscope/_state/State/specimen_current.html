state.specimen_current

Provides the status of the specimen current.

Properties

is_valid [Read only] Indicates the calibration status of the specimen current measurement. If True, the system is calibrated and the value returned by value is accurate. If False, the value is inaccurate.
value [Read only] Returns the actual specimen current in amperes.

Methods

calibrate() Performs the specimen current offset calibration routine. This function is only available on the Axia ChemiSEM system.
