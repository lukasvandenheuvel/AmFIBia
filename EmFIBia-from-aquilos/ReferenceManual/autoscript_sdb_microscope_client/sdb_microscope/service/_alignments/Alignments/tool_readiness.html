service.alignments.tool_readiness

Provides access to the ToolReadiness service, which monitors the status of microscopic alignments and allows running selected alignments either individually or in a sequence.

Properties

is_installed [Read only] Returns True if the ToolReadiness service is installed and running.
is_running [Read only] Returns True if the ToolReadiness service is executing any alignment.

Methods

abort() Aborts all running alignments that were initiated asynchronously by calling the @Service.Alignments.ToolReadiness.StartAlignment or start_direct_alignment() functions.
get_alignment_status() Returns the status of the provided alignment procedures.
get_available_alignments() Retrieves the list of available alignments in the ToolReadiness service.
get_available_direct_alignments() Retrieves the list of direct alignments available in the ToolReadiness service.
run_alignments() Runs one or more alignment procedures synchronously. The function call will return only after all procedures have been completed.
run_direct_alignment() Runs a direct alignment synchronously. The function call will return only when the alignment has finished.<summary> </summary> <param name="alignment">An object that specifies which direct alignment should be run.</param> <returns>A data structure that provides status information for the completed alignment.</returns>
start_alignments() Runs one or more alignment procedures asynchronously. The call initiates the procedures and returns without waiting for their completion.
start_direct_alignment() Runs a direct alignment asynchronously. The call initiates the procedure and returns without waiting for its completion.
