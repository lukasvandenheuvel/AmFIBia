service.alignments.cap_probe

Properties

offset Gets or sets the Cap Probe offset X and Y. The value can only be set after the system has been switched to service mode.

Methods

add_calibration() Sets the Cap Probe scale factor. This function can only be called after the system has been switched to service mode.
get_calibration() Gets the Cap Probe scale factor.
get_statistics() Reads N samples from the Cap Probe and returns the mean value, standard deviation and value range.
get_trained_value() Returns the Cap Probe set point for a stage tilt angle.
read_value() Reads one value from the Cap Probe.
update_trained_value() Updates the Cap Probe set point for a stage tilt angle. This function can only be called after the system has been switched to service mode.
