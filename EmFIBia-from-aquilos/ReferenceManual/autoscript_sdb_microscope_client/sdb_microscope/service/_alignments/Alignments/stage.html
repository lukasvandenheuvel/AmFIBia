service.alignments.stage

Properties

center_of_rotation Reads or saves the center of rotation of the bulk stage. The value can only be set after the system has been switched to service mode.
current_coarse_position [Read only] Returns the current position of the coarse stage motors. Supported only on systems equipped with a piezo stage.
current_fine_position [Read only] Returns the current position of the fine stage motors. Supported only on systems equipped with a piezo stage.
home_offset Reads or saves the home offset of stage axes. The value can only be set after the system has been switched to service mode.

Methods

recording_start_trace() Starts the stage recording trace.
recording_stop_trace() Stops the stage recording trace.
