service.autoscript.client.is_stop_requested

Returns True if a graceful stop of the running script has been requested.

Data type

[Read only] bool

Remarks

The request for an execution stop can be made from the AutoScript Runner application or from the PyCharm editor where AutoScript extensions are installed.

Examples


        for i in range(100):
            # Allow the user to stop the cycle at a convenient time
            if microscope.service.autoscript.client.is_stop_requested:
                # If stop is requested, bring the stage to the center and turn off electron beam
                print("Stopping experiment...")
                microscope.specimen.stage.absolute_move(StagePosition(x=0, y=0))
                microscope.beams.electron_beam.turn_off()
        
                # Abort script execution by raising KeyboardInterrupt exception
                microscope.service.autoscript.client.check_stop_requested()
        
            # Acquire an image
            image = microscope.imaging.grab_frame()
            print(f"Acquisition number: {i}")
