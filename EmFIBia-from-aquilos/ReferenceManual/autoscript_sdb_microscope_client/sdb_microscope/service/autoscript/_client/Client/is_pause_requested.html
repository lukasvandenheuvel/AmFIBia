service.autoscript.client.is_pause_requested

Returns True if a pause of the running script has been requested.

Data type

[Read only] bool

Remarks

The request for an execution pause can be made from the AutoScript Runner application or from the PyCharm editor where AutoScript extensions are installed.

The call to is_pause_requested is non-blocking, unlike the call to check_pause_requested(), which blocks until the pause signal is cleared by pressing the Resume button in AutoScript Runner or PyCharm.

Examples


        for i in range(100):
            # Acquire an image
            image = microscope.imaging.grab_frame()
            print(f"Acquisition number: {i}")
        
            # Check if execution pause is requested
            if microscope.service.autoscript.client.is_pause_requested:
                # Do some action before actually pausing the script
                # ...
                
                # Pause script execution until Resume is pressed
                microscope.service.autoscript.client.check_pause_requested()
