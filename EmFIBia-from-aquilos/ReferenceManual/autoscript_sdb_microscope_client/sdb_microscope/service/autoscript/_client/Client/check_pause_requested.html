service.autoscript.client.check_pause_requested

Checks if a pause has been requested for the running script. If so, execution of the script is blocked until the user presses the Resume button.

Syntax

check_pause_requested()

Remarks

The request for an execution pause can be made from the AutoScript Runner application or from the PyCharm editor where AutoScript extensions are installed.

Calling check_pause_requested() blocks execution until the pause signal is cleared by pressing the Resume button in AutoScript Runner or PyCharm.
If check_pause_requested() is called but no pause request has been created, the script continues without pausing.

Examples


        for i in range(100):
            # Acquire an image
            image = microscope.imaging.grab_frame()
            print(f"Acquisition number: {i}")
        
            # Pause script execution if requested
            microscope.service.autoscript.client.check_pause_requested()
