service.autoscript.client.get_preferred_unit

Gets the preferred unit for a specified quantity.

Syntax

get_preferred_unit(quantity)

Parameters

quantity str The quantity for which you want to obtain your preferred unit. You can use Quantity enumeration.

Return value

str The preferred unit for a specified quantity. Maps to Unit enumeration.

Remarks

See the description of the preferred unit concept in service.autoscript.client.set_preferred_unit().

Examples


        # Print preferred unit for API functions dealing with temperature
        print(microscope.service.autoscript.client.get_preferred_unit(Quantity.TEMPERATURE))
