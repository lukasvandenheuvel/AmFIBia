service.alignments.tool_readiness.get_alignment_status

Returns the status of the provided alignment procedures.

Syntax

get_alignment_status(alignments, use_case_settings)

Parameters

alignments List[ToolReadinessAlignment] A list of ToolReadinessAlignment objects that specify the alignments for which the status should be determined.
use_case_settings ToolReadinessUseCaseSettings Additional settings that specify the system conditions for which the alignment status should be determined.

Return value

List[ToolReadinessAlignmentStatus] A list of structures that provide detailed status information for the completed alignments.
