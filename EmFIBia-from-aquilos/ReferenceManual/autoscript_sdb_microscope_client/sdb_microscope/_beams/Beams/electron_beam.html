beams.electron_beam

Provides control and status of the electron beam.

Classes

angular_correction Enables control and provides the status of the angular correction.
beam_current Enables control and provides the status of the electron beam current.
beam_deceleration Enables control and provides the status of the electron beam deceleration.
beam_mode Enables control and provides the status of the electron beam mode.
beam_shift Enables control and provides the status of the beam shift.
crossover_zoom Enables control and provides the status of the crossover zoom parameter on the electron beam.
emission_current Provides the status of the electron beam emission current.
high_voltage Enables control and provides the status of the acceleration voltage.
horizontal_field_width Enables control and provides the status of the horizontal field width.
lens_alignment Enables control and provides the status of the electron beam lens alignment.
optical_mode Enables control and provides the status of the electron beam optical mode.
protective_shutter Functions related to the retractable shutter that protects the SEM column during FIB milling.
scanning Enables control provides the status of beam scanning properties.
source Enables control and provides the status the electron beam source.
source_tilt Enables control and provides the status of the electron beam source tilt.
stigmator Enables control and provides the status of the stigmator.
working_distance Enables control and provides the status of the working distance.

Properties

is_blanked [Read only] Gets the current blank state of the beam.
is_installed [Read only] Tells whether the electron beam is installed on the system.
is_on [Read only] Returns True if the beam is turned on.

Methods

blank() Blanks the beam.
turn_off() Turns the beam off.
turn_on() Turns the beam on.
unblank() Unblanks the beam.
