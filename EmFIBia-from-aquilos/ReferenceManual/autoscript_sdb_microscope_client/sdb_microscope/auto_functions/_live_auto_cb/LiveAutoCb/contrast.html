auto_functions.live_auto_cb.contrast

Enables control of detector contrast in live adjustment mode.

Properties

limits [Read only] Retrieves the valid range of values for the live mode contrast parameter.
value Sets or retrieves the live mode contrast level for the active view.

Examples


    	# Set live contrast to 0.1 in the active view
    	microscope.auto_functions.live_auto_cb.contrast.value = 0.1      
    

    	# Read live contrast limits
    	print(microscope.auto_functions.live_auto_cb.contrast.limits)
