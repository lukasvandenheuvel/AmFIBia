imaging.lut.optimize

Adjusts digital contrast, brightness, and gamma to achieve an ideal histogram stretch for the image in the active view.

Syntax

optimize()

Remarks

  • The function calculates and sets an optimal LUT (Look-Up Table) based on the current image in the active view, ensuring an ideal histogram distribution. This setting is saved for the specific combination of the active view and imaging device. The new LUT will be applied to all subsequent image acquisitions in that view until it is reset.
  • If the image characteristics change, for instance, when the stage is moved to a new location, it is advisable to re-run the function to adjust the LUT for the new image.
  • In the Microscope Control application, this function is referred to as "Auto Levels".
