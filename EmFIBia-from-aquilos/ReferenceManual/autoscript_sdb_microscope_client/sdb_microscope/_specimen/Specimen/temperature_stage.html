specimen.temperature_stage

Enables control and provides the status of the microscope temperature stage.

Classes

humidity Enables control of humidity of wet samples.
ramping_speed Provides control and status of the temperature ramping speed.
temperature Enables control and provides the status of the stage temperature.

Properties

is_installed [Read only] Tells whether the microscope support temperature stages. This property does not indicate whether a temperature stage is currently connected.
is_on [Read only] Returns the on/off state of the currently connected temperature stage.
type [Read only] Returns the type of the currently connected temperature stage. Maps to TemperatureStageType enumeration.

Methods

ramp() Ramps the stage temperature up or down according to the settings specified by the parameter.
select_heat_shield() Selects the heat shield for the currently connected temperature stage.
turn_off() Turns the temperature stage off.
turn_on() Turns the temperature stage on.
