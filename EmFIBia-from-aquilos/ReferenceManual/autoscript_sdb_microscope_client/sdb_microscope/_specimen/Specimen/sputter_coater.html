specimen.sputter_coater

Allows monitoring and control of the sputter coater device, commonly referred to as "MicroSputter".

Classes

current Enables control and provides the status of the sputter coater current.

Properties

is_homed [Read only] Returns True if the sputter coater device has been homed.
is_inserted [Read only] Returns True if the sputter coater is inserted.
is_installed [Read only] Returns True if the sputter coater device is present on the microscope.

Methods

home() Runs the home procedure, which calibrates the position of the sputter coater device.
prepare() Prepares the microscope for the sputtering process.
recover() Returns the system to the state before the prepare() function was called.
run() Executes the sputtering procedure.
