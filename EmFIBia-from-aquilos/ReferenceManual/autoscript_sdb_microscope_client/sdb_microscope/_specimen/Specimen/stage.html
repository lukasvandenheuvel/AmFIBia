specimen.stage

Enables control and provides the status of the bulk stage.

Classes

holder Provides access to the specimen holder configuration.
safety_settings Enables control of the stage safety settings.

Properties

current_position [Read only] Retrieves the current stage position.
is_homed [Read only] Returns True if the stage is homed.
is_installed [Read only] Returns True if the microscope is equipped with the bulk stage.
is_linked [Read only] Returns True if the stage Z is linked to the free working distance.

Methods

absolute_move() Moves the stage to the specified position.
get_axis_limits() Returns limits of a stage axis in meters or radians.
home() Homes the specified axes of the stage. If no axis is specified, all axes are homed.
link() Links the stage Z to the free working distance.
load_sample_map() Loads the definition of a sample map from a file.
move_to_device() Moves the stage to a position under an imaging device. In that position imaging with the device is allowed.
relative_move() Moves the stage by the specified delta position.
set_default_coordinate_system() Sets the default coordinate system of the stage.
set_default_move_settings() Sets the default settings for stage moves. Subsequent absolute_move() and relative_move() calls where MoveSettings are not specified use these settings.
start_jogging() Initiates the continuous movement ("jogging") of the stage, determined by the input velocity in m/s for each axis. The stage jogs until stopped or until the time interval is reached. The function is asynchronous and returns immediately.
stop() Immediately stops movement of the stage in all axes.
unlink() Unlinks the stage Z from the free working distance.
