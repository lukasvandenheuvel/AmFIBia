specimen.manipulator

Enables control and provides the status of the manipulator device. The device is also referred to as EasyLift.

Properties

current_position [Read only] Gets the current manipulator position.
is_installed [Read only] Tells whether the manipulator device is installed on the microscope.
state [Read only] Returns the insertion state of the manipulator needle. Possible states are described by the ManipulatorState enumeration.

Methods

absolute_move() Moves the manipulator to the specified position.
get_saved_position() Returns a saved (named) manipulator position. There are four default named positions. The user can save more using the main user interface, but these positions are inaccessible to AutoScript.
insert() Inserts the manipulator either to a default or specified position.
relative_move() Moves the manipulator by the specified delta position.
retract() Retracts the manipulator.
set_default_coordinate_system() Establishes the default coordinate system for the manipulator. If no system is explicitly specified by the position, any subsequent calls to absolute_move(), relative_move() and insert() will utilize this default system.
