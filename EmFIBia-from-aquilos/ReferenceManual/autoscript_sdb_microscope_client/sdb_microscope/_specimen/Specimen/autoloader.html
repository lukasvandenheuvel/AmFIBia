specimen.autoloader

Represents the Autoloader device.

Classes

stage Represents the stage holder on the Autoloader device.

Properties

is_installed [Read only] Returns True if the Autoloader is present on the microscope, False otherwise.

Methods

dock() Docks the Autoloader cassette.
get_compartment() Returns an object representing the specified cryo compartment of the Autoloader device.
get_slots() Returns a list of slots that are available on the cassette.
load() Loads the sample from the specified slot of the cassette to the stage holder.
undock() Undocks the Autoloader cassette.
unload() Unloads the sample from the stage holder and returns it to either its original slot in the cassette or to the first available empty slot, unless a slot identifier is specified.
