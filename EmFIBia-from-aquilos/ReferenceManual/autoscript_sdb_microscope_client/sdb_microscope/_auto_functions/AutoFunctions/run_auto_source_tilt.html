auto_functions.run_auto_source_tilt

Runs a routine that automatically adjusts the source tilt in the active view.

Syntax

run_auto_source_tilt()
run_auto_source_tilt(settings)

Parameters

settings RunAutoSourceTiltSettings Settings for the automatic source tilt adjustment routine.

Remarks

This routine only works with the electron beam, which must be selected as the imaging device in the active view.
There is only one method available for the routine, Volumescope, which has to be declared explicitly because of its specialized nature.

Examples


        # Run the auto source tilt routine in "Volumescope" variant, providing additional parameters
        settings = RunAutoSourceTiltSettings(method="Volumescope", dwell_time=1e-6)
        microscope.auto_functions.run_auto_source_tilt(settings)
