auto_functions.run_auto_focus

Runs the automatic focus routine in the active view.

Syntax

run_auto_focus()
run_auto_focus(settings)

Parameters

settings RunAutoFocusSettings Settings for the automatic focus routine.

Remarks

  • Two algorithm variants ("methods") are available for the routine: Standard and Volumescope. You can select the desired variant using the method property in the RunAutoFocusSettings structure. If not specified, the Standard algorithm variant is executed.
  • The standard method performs the same algorithm that you can run manually from the main microscope user interface.

The following table lists the applicable settings for each of the available methods. Note that the "Standard" method has a different set of applicable settings depending on the system.

Standard on Systems 1) Standard on Systems 2) Standard on Systems 3) Volumescope
dwell_time NO YES NO YES
horizontal_field_width NO NO NO YES
line_integration NO NO NO YES
number_of_frames NO NO NO YES
reduced_area NO NO YES YES
resolution NO NO NO YES
working_distance_step NO NO NO YES
display_point YES YES NO NO
maximum_iteratons NO NO NO YES

Systems 1) ... Apreo ChemiSEM (37.x)
Systems 2) ... Axia ChemiSEM (platform software version 27.x), Quattro (35.x)
Systems 3) ... All other systems

Examples


        # Run the standard auto focus routine
        microscope.auto_functions.run_auto_focus()
        
        # Run the standard auto focus routine, specifying the reduced area location
        settings = RunAutoFocusSettings(reduced_area=Rectangle(0.1, 0.1, 0.3, 0.3))
        microscope.auto_functions.run_auto_focus(settings)
        
        # Run the standard auto focus routine, specifying the display point
        settings = RunAutoFocusSettings(display_point=Point(0.4, 0.5))
        microscope.auto_functions.run_auto_focus(settings)

        # Run the auto focus routine in "Volumescope" algorithm variant
        settings = RunAutoFocusSettings(method="Volumescope")
        microscope.auto_functions.run_auto_focus(settings)
        
        # Run the auto focus routine in "Volumescope" algorithm variant, providing additional parameters
        settings = RunAutoFocusSettings(method="Volumescope", resolution="1536x1024",
                                        dwell_time=500e-9, reduced_area=Rectangle(0.1, 0.1, 0.8, 0.02))
        microscope.auto_functions.run_auto_focus(settings)
