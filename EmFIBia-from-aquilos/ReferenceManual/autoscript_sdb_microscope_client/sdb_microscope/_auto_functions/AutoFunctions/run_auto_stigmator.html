auto_functions.run_auto_stigmator

Runs a routine that automatically optimizes stigmator in the active view.

Syntax

run_auto_stigmator()
run_auto_stigmator(settings)

Parameters

settings RunAutoStigmatorSettings Settings for the automatic stigmator adjustment routine.

Remarks

  • Two algorithm variants ("methods") are available for the routine: Standard and OngEtAl. You can select the desired routine via property method of the RunAutoStigmatorSettings structure. Standard method is the default one.
  • On Axia family microscopes the Standard method accepts two optional parameters in the settings structure.
  • The Standard method performs the same algorithm that you can also run manually from the microscope's main user interface.
  • The OngEtAl algorithm optimizes both the stigmator and focus.

The following table lists applicable settings for each method:

Standard Standard on selected systems *) OngEtAl
dwell_time NO YES YES
horizontal_field_width NO NO YES
line_integration NO NO YES
number_of_frames NO NO YES
reduced_area NO NO YES
resolution NO NO YES
stigmation_step NO NO YES
display_point NO YES NO
maximum_iteratons NO NO YES

*) The Standard autostigmator method on Axia ChemiSEM systems (software platform version 27.x) and Apreo ChemiSEM (platform version 37.x) has a different set of applicable settings.

Examples


        # Run the standard auto stigmator routine
        microscope.auto_functions.run_auto_stigmator()
    
        # Run the "OngEtAl" auto stigmator routine
        settings = RunAutoStigmatorSettings(method="OngEtAl")
        microscope.auto_functions.run_auto_stigmator(settings)
    
        # Run the "OngEtAl" auto stigmator routine, supplying additional parameters
        settings = RunAutoStigmatorSettings(method="OngEtAl", resolution="192x128", reduced_area=Rectangle(0.1, 0.1, 0.8, 0.8))
        microscope.auto_functions.run_auto_stigmator(settings)
    
        # Run the standard auto stigmator routine with the Axia specific parameter from which the position of reduced area is calculated
        settings = RunAutoStigmatorSettings(display_point=Point(0.2, 0.3))
        microscope.auto_functions.run_auto_stigmator(settings)
