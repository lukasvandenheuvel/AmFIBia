auto_functions.run_auto_stigmator_centering

Runs a routine that automatically optimizes stigmator centering in the active view.

Syntax

run_auto_stigmator_centering()
run_auto_stigmator_centering(settings)

Parameters

settings RunAutoStigmatorCenteringSettings Settings for the automatic stigmator centering adjustment routine.

Remarks

This routine only works with the electron beam, which must be selected as the imaging device in the active view.
There is only one method available for the routine, Volumescope, which has to be declared explicitly because of its specialized nature.

Examples


    # Run the auto stigmator centering routine in "Volumescope" algorithm variant 
    microscope.auto_functions.run_auto_stigmator_centering(RunAutoStigmatorCenteringSettings(method="Volumescope"))

    # Run the auto stigmator centering routine, specifying position of the reduced area
    settings = RunAutoStigmatorCenteringSettings(method="Volumescope", reduced_area=Rectangle(0.4, 0.4, 0.2, 0.2))
    microscope.auto_functions.run_auto_stigmator_centering(settings)
