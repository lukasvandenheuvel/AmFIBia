auto_functions.run_auto_cb

Runs a routine that automatically optimizes contrast and brightness of the active detector in the active view.

Syntax

run_auto_cb()
run_auto_cb(settings)

Parameters

settings RunAutoCbSettings Settings for the automatic brightness and contrast adjustment routine.

Remarks

Two algorithm variants ("methods") are available for the routine: Standard and MaxContrast. You can select the desired routine via the property method of the RunAutoCbSettings structure. Standard method is the default one.

In the following table you will find the relevant settings for each of the available methods:

Standard MaxContrast
black_target YES NO
white_target YES NO
calibrate_detector NO YES
dwell_time NO YES
line_integration NO YES
max_black_clipping NO YES
max_white_clipping NO YES
number_of_frames NO YES
resolution NO YES
brightness NO YES

Examples


    # Run the standard auto contrast/brightness routine 
    microscope.auto_functions.run_auto_cb()

    # Run the auto contrast/brightness routine using a specialized "MaxContrast" method 
    settings = RunAutoCbSettings(method="MaxContrast")
    microscope.auto_functions.run_auto_cb(settings)

    # Run the auto contrast/brightness routine using a specialized MaxContrast method and provide custom settings 
    settings = RunAutoCbSettings(method="MaxContrast", resolution="192x128", calibrate_detector=True)
    microscope.auto_functions.run_auto_cb(settings)
