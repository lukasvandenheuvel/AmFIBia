vacuum.pressure_limiting_aperture.value

Returns the name of the currently mounted pressure-limiting aperture (PLA).

Data type

[Read only] str

Remarks

This property is read-only. To select the currently mounted PLA you must call the vacuum.chamber_pressure.pump() function and specify the PLA type in the VacuumSettings parameter.

Examples


		# Read currently selected PLA
		pla_type = microscope.vacuum.pressure_limiting_aperture.value

		# Switch to low vacuum mode and select "XRayCone" PLA
		settings = VacuumSettings(mode=VacuumMode.LOW_VACUUM, pressure=10, pressure_limiting_aperture="XRayCone")
		microscope.vacuum.pump(settings)
