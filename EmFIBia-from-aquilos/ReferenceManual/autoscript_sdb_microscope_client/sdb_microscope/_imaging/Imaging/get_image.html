imaging.get_image

Retrieves a microscope image currently present in the active view.

Syntax

get_image()
get_image(settings)

Parameters

settings GetImageSettings Additional settings that determine how the image is retrieved when the active view is in acquiring state.

Return value

AdornedImage The microscope image retrieved from the active view.

Remarks

  • The method does not initiate the scanning of a new image frame. It inspects the imaging view and returns the image stored there from previous calls to grab_frame().
    If you want to acquire a new image with the active imaging device, call grab_frame() instead.
  • When the wait_for_frame flag in settings is set to True and the view is in an acquiring state, the get_image() function will block until the end of the frame is reached, and then it returns the image. If the view is in an idle state, the wait_for_frame flag is disregarded.
  • The wait_for_frame flag in settings is applicable only when the active device is either the electron or ion beam.

Examples


        # Read image from active view and save it to disk; the get_image() function doesn't block
        image = microscope.imaging.get_image()
        image.save("c:\\temp\\image.tif")
    

        # The following example showcases live SEM image streaming,
        # with completed image frames being displayed in a window as they are scanned.
        
        import cv2
        
        # Start scanning with electron beam
        microscope.imaging.set_active_device(ImagingDevice.ELECTRON_BEAM)
        microscope.imaging.start_acquisition()
    
        while microscope.imaging.state == ImagingState.ACQUIRING:
            # Wait for the completion of one frame and then copy the image
            settings = GetImageSettings(wait_for_frame=True)    
            image = microscope.imaging.get_image(settings)
        
            # Show the image in a window
            cv2.imshow('SEM Image', image.data)
        
            # Abort the loop on Esc key press
            k = cv2.waitKey(10)
            if k == 27:
                break
