imaging.grab_multiple_frames

Scans new frames in all compatible views and returns the acquired images.

Syntax

grab_multiple_frames()
grab_multiple_frames(settings)

Parameters

settings GrabFrameSettings The custom scan settings to use instead of the default values.

Return value

List[AdornedImage] The list of microscope images, each corresponding to a single compatible view.

Remarks

  • The function initiates scanning of new images and waits until they are completely scanned.
  • If views are not specified in settings, the scan is performed in all views compatible with the active view, i.e. in views with the same imaging device.
  • In settings you can specify the exact list of views that should participate in the grab multiple frames action. Specifying incompatible views will result in an error.
  • Views that have incompatible detectors assigned to them cannot participate in the grab multiple frames action. The action is started but results in an error. Which detectors are compatible, i.e. can simultaneously generate signal, cannot be determined via the AutoScript API.
  • Custom scan settings can also be provided by the settings parameter. If not specified, the actual scan settings are determined as follows:
    • resolution, dwell_time and reduced_area will default to the current settings in the active view.
    • Other settings will fallback to predefined constant default values:
      • bit_depth will be 8.
      • line_integration will be 1.
      • scan_interlacing will be 1.
      • frame_integration will be 1.
      • drift_correction will be False.
  • The grab_multiple_frames() function can perform image acquisition only in standard resolutions, that is resolutions from the ScanningResolution enumeration. Maximum resolution is therefore 6144x4096. An error is returned for other resolutions.
  • Parameter preview_resolution, if specified in settings, is ignored by grab_multiple_frames().

Examples


        settings = GrabFrameSettings(resolution=ScanningResolution.PRESET_1536X1024, frame_integration=4)
        images = microscope.imaging.grab_multiple_frames(settings)
    
        print(f"Acquired {len(images)} images.")
    

        # Acquire multiple images by integrating 8 consecutive frames in all compatible views and correcting them for drift
        settings = GrabFrameSettings(resolution=ScanningResolution.PRESET_1536X1024, dwell_time=100e-9,
                                     frame_integration=8, drift_correction=True)
        images = microscope.imaging.grab_multiple_frames(settings)
    

        # Acquire images from views 1, 2 and 3
        settings = GrabFrameSettings(views=[1, 2, 3])
        images = microscope.imaging.grab_multiple_frames(settings)
        
        # 3 images returned
    

	# Grab multiple frames with smart frame integration.
	# Create grab frame settings
	settings = GrabFrameSettings(smart_frame_integration=True, frame_integration=16, dwell_time=3e-7)
	
	# Grab image
	image = microscope.imaging.grab_multiple_frames(settings)
