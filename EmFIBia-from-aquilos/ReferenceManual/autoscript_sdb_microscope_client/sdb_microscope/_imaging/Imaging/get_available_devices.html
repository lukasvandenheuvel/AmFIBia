imaging.get_available_devices

Returns a list of imaging devices installed on the microscope.

Syntax

get_available_devices()

Return value

List[int] The list of numerical IDs for the imaging devices installed on the microscope. To convert these IDs into a readable format, use the ImagingDevice enumeration.

Examples


        # Read installed imaging devices
        devices = microscope.imaging.get_available_devices()
        
        # Returned list: [1, 2, 4, 6]
        
        # Print the list in readable format
        for device in devices:
            print(ImagingDevice.explain(device))
