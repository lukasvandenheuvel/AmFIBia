imaging.stop_acquisition

Stops live image acquisition in the active view.

Syntax

stop_acquisition()
stop_acquisition(stop_acquisition_method)

Parameters

stop_acquisition_method int Specifies how the stop operation should be performed. You can provide values from the StopAcquisitionMethod enumeration.

Remarks

  • The parameterless version of the function stops imaging immediately (equivalent to StopAcquisitionMethod.ABORT).
  • If image acquisition is already stopped, the function does nothing.
  • The stop_acquisition_method parameter is applicable only to scanning imaging devices, that is electron and ion beams. For camera-type devices it is ignored.

Examples


        # Prepare view 1 for imaging with electron beam
        microscope.imaging.set_active_view(1)
        microscope.imaging.set_active_device(ImagingDevice.ELECTRON_BEAM)
        
        # Initialize scanning conditions
        microscope.beams.electron_beam.scanning.resolution.value = ScanningResolution.PRESET_1536X1024
        microscope.beams.electron_beam.scanning.dwell_time.value = 10e-6
        
        # Scan one frame
        microscope.imaging.start_acquisition()
        microscope.imaging.stop_acquisition(StopAcquisitionMethod.END_OF_FRAME)
