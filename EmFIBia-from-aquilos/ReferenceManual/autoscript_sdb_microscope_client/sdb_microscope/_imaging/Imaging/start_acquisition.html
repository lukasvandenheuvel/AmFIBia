imaging.start_acquisition

Starts live image acquisition with the active device in the active view.

Syntax

start_acquisition()

Remarks

  • The function starts live imaging with the active imaging device in the active view. The device continues image acquisition until imaging.stop_acquisition() is called.
  • If imaging is already started, the function does nothing.

Examples


        # Prepare view 1 for imaging with electron beam
        microscope.imaging.set_active_view(1)
        microscope.imaging.set_active_device(ImagingDevice.ELECTRON_BEAM)
        
        # Initialize scanning conditions
        microscope.beams.electron_beam.scanning.resolution.value = ScanningResolution.PRESET_1536X1024
        microscope.beams.electron_beam.scanning.dwell_time.value = 10e-6
        
        # Start scanning with the electron beam
        microscope.imaging.start_acquisition()
