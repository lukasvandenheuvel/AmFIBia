vacuum.vent

Vents the specimen chamber.

Syntax

vent()

Remarks

  • Calling vent() when the chamber is already vented or is in the "venting" transition state does nothing.
  • Both pump() and vent() functions are synchronous, i.e. they return when the change in vacuum state is complete. However, if the vacuum system is already in the "pumping" transition state when the vent() function is called, the function will interrupt the transition in progress and start venting the chamber.
