detector.retract

Retracts the active detector unless a specific detector type is provided in settings.

Syntax

retract()
retract(settings)

Parameters

settings DetectorRetractSettings The additional settings for the retract operation.

Remarks

  • Retracting a detector that is already retracted will have no effect.
  • Attempting to retract a non-retractable detector will throw an exception. You can check if a detector is retractable by calling detector.get_info().
  • The parameterless version of the function operates on the active detector in the active view. If a detector type is specified in the settings, the function will retract it without needing to activate it in the view.
  • In the current version, AutoScript does not support image acquisition with the "T-PIX" detector or the ability to change its insertion state.

Examples


        # Retract the active detector
        microscope.detector.retract()
    

        # Retract ABS detector
        settings = DetectorRetractSettings(type=DetectorType.ABS)
        microscope.detector.retract(settings)
    

        # Retract all inserted detectors
        installed_detectors = microscope.detector.type.all_values
        for detector_type in installed_detectors:
            detector_info = microscope.detector.get_info(detector_type)
            if detector_info.is_retractable and detector_info.can_retract:
                print("Retracting", detector_type)
                microscope.detector.retract(DetectorRetractSettings(type=detector_type))
