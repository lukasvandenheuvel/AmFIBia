detector.custom_settings

Additional detectors settings.

Classes

converter_voltage Enables control and provides the status of the active detector converter voltage.
grid_voltage Enables control and provides the status of the grid voltage for the active detector.
md_shutter Represents the Mirror Detector Shutter.
mirror_voltage Enables control and provides the status of the active detector mirror voltage.
scintillator_voltage Enables control and provides the status of the active detector scintillator voltage.
suction_tube_voltage Enables control and provides the status of the suction tube voltage for the active detector.

Methods

get_segment_polarity() Gets the polarity of a detector segment.
list_all_segments() Returns all segments available on the active detector.
reset_segments() Resets all eligible segments of the active detector while it is switched to custom mode.
set_segment_polarity() Sets the polarity of a detector segment.

Remarks

Similar to other API items within the detector API group, the properties listed above affect an active detector within the active view. However, not all settings are relevant to all detector types. The following table provides a summary of which settings are applicable to specific detector types.
