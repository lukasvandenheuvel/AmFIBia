detector.insert

Inserts the active detector unless a specific detector type is provided in settings.

Syntax

insert()
insert(settings)

Parameters

settings DetectorInsertSettings The additional settings for the insert operation.

Remarks

  • Inserting a detector that is already inserted will have no effect.
  • Attempting to insert a non-retractable detector will throw an exception. You can check if a detector is retractable by calling detector.get_info().
  • The parameterless version of the function operates on the active detector in the active view. If a detector type is specified in the settings, the function will insert it without needing to activate it in the view.
  • In the current version, AutoScript does not support image acquisition with the "T-PIX" detector or the ability to change its insertion state.

Examples


        # Insert the active detector
        microscope.detector.insert()    
    

        # Assuming STEM4 is the active detector, insert it into the standard position
        settings = DetectorInsertSettings(insert_position=Stem4Positions.STANDARD)
        microscope.detector.insert(insert_settings)    
    

        # Insert STEM3 detector
        insert_settings = DetectorInsertSettings(type=DetectorType.STEM3)
        microscope.detector.insert(insert_settings)
    

	# Insert EDS detector to standard position
	insert_settings = DetectorInsertSettings(type=DetectorType.EDS, insert_position=EdsDetectorPositions.STANDARD_INSERTED)
	microscope.detector.insert(insert_settings)       
    

	# Insert EDS detector to semi retracted position
	insert_settings = DetectorInsertSettings(type=DetectorType.EDS, insert_position=EdsDetectorPositions.SEMI_RETRACTED)
	microscope.detector.insert(insert_settings)
