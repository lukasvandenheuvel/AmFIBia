detector.get_info

Provides details on the detector's physical properties and current state.

Syntax

get_info()
get_info(detector_type)

Parameters

detector_type str The type of the detector.

Return value

DetectorInfo A DetectorInfo structure containing the detector's state and physical properties.

Remarks

The state information returned is valid only at the time the get_info() function is called. Accessing properties from the returned DetectorInfo structure later will provide the cached state, not the current microscope state.

Examples


        # Find all retractable detectors installed on the microscope
        installed_detectors = microscope.detector.type.all_values
        retractable_detectors = [detector for detector in installed_detectors if microscope.detector.get_info(detector).is_retractable]
