detector.camera_settings

Contains settings of several camera detectors - Optical microscope (HiResOptical), Fluorescence light microscope (FLM) and others.

Classes

background_correction Enables you to eliminate noise or dirt present on the camera glass from the live image.
binning Allows you to access the binning parameter of the active camera detector.
bit_depth Allows you to access the bit depth parameter of the active detector.
bright_field Allows you to access the bright field parameter of the active detector.
dark_field Allows you to access the dark field parameter of the active detector.
emission Allows you to access the emission parameter of the active camera detector.
exposure_time Allows you to access the exposure time parameter of the active detector.
filter Allows you to access the filter parameter of the active camera detector.
focus Allows you to access the focus parameter of the active camera detector.

Properties

color Gets or sets the color correction for the active detector. Color channels are set within the range of [0, 1] corresponding to 0% - 100%. The alpha channel is ignored.

Remarks

Similar to other API items within the detector API group, the properties listed above affect an active camera detector within the active view. However, not all settings are relevant to all camera types. The following table provides a summary of which settings are applicable to specific camera types.
