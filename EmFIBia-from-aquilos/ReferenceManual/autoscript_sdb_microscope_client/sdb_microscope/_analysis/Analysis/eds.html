analysis.eds

Provides functions for chemical microanalysis of the sample using an energy dispersive spectroscopy (EDS) detector.

Classes

mapping Functions related to the generation of EDS elemental and phase maps.

Properties

is_installed [Read only] Indicates whether the EDS detector is installed on the system.

Methods

acquire_spectrum() Scans the specified region and collects EDS spectrum data from it.
get_composition() Performs quantification of chemical elements in the specified EDS spectrum.
get_peak_id() Identifies the presence of chemical elements in the given EDS spectrum.
get_spectrum() Returns the EDS spectrum collected since the last reset of the EDS spectrum buffer.
reset_spectrum() Clears the buffer containing EDS spectrum data.

Remarks

Certain new systems, such as the Apreo ChemiSEM (xT 37.x), may feature a retractable EDS detector. Ensure that the EDS detector is properly inserted; otherwise, the EDS spectrum will be empty.



# Insert EDS detector
insert_settings = DetectorInsertSettings(type=DetectorType.EDS, insert_position=EdsDetectorPositions.STANDARD_INSERTED)
microscope.detector.insert(insert_settings)
