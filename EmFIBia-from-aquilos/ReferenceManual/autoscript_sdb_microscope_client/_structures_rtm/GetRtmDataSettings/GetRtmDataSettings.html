GetRtmDataSettings

Contains settings for the function get_data().

Items

pattern_ids [Optional] List[int] Specifies from which patterns to retrieve Real Time Monitor data. If the list is empty, all patterns are used.
wait_for_next_data [Optional] bool When True, the function get_data() waits until the next data is available. If no data was collected and the patterning job ended, None is returned. The flag can be useful for patterning jobs where one pass takes long and the user does not want to process the same data twice.
