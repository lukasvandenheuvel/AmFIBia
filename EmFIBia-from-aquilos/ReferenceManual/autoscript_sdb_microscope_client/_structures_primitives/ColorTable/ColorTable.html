ColorTable

An auxiliary class providing a selection of named colors that align with the color scheme used in the main microscope user interface for rendering EDS elemental and phase maps.

These colors are defined:

RED 0xf20000  
DARK_ORANGE 0xf43100  
LIGHT_ORANGE 0xf5a500  
GOLD 0xffdd00  
YELLOW 0xffff00  
LIME 0xccff00  
LIGHT_GREEN 0x82e000  
DARK_GREEN 0x40e20a  
TEAL 0x00ff9d  
CYAN 0x3cf2e9  
LIGHT_BLUE 0x00d4ff  
ROYAL_BLUE 0x0091f2  
DARK_BLUE 0x0031e5  
PURPLE 0x6a1fe2  
VIOLET 0xaf00e5  
MAGENTA 0xef00e3  
PINK 0xf4009f  
BORDEAUX 0xe51673
