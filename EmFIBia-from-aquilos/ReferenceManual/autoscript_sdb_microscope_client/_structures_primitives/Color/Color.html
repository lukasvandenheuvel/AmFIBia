Color

Represents the RGBA color model. The alpha channel controls the transparency of the color, while the other channels determine the color intensity. Values are in the range of [0, 1] corresponding to 0% - 100%.

Items

red [Optional] float
green [Optional] float
blue [Optional] float
alpha [Optional] float

Methods

create_from_hex_color() Creates an instance of the Color class and initializes it using RGB values encoded within a single integer value.
create_from_rgb256() Creates an instance of the Color class and initializes it using RGB values.
get_as_hex_color() Encodes the color into a single integer value in 0xRRGGBB format where RR, GG, and BB are bytes representing red, green, and blue color components in the range 0 to 255.
