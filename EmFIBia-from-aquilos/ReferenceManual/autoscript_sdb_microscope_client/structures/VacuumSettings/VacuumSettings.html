VacuumSettings

Contains settings for the function pump().

Items

mode [Optional] str The mode of specimen chamber vacuum. Accepts values from the VacuumMode enumeration.
gas [Optional] str The type of gas to be used for the Low Vacuum or ESEM modes. Accepts values from the VacuumGasType enumeration.
pressure [Optional] float The target chamber pressure.
pressure_limiting_aperture [Optional] str The name of the pressure-limiting aperture (PLA) that is mounted on the column. The available_values property can be used to read out apertures that are known to the system and applicable in the current state of the microscope.
