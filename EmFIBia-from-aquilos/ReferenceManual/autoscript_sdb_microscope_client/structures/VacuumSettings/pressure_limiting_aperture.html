VacuumSettings.pressure_limiting_aperture

The name of the pressure-limiting aperture (PLA) that is mounted on the column. The available_values property can be used to read out apertures that are known to the system and applicable in the current state of the microscope.

Data type

[Optional] str

Examples


		# Switch to low vacuum mode and select "XRayCone" PLA
		settings = VacuumSettings(mode=VacuumMode.LOW_VACUUM, pressure=10, pressure_limiting_aperture="XRayCone")
		microscope.vacuum.pump(settings)
