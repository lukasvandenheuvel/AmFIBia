RunAutoStigmatorCenteringSettings

Contains settings for the automatic adjustment of stigmator center.

Items

method [Optional] str The name of the auto stigmator centering algorithm variant.
dwell_time [Optional] float The scanning dwell time to apply during the routine.
line_integration [Optional] int The number of lines to integrate when acquiring image in each auto function iteration.
number_of_frames [Optional] int The number of frames to integrate when acquiring image in each auto function iteration.
reduced_area Get: [Optional] Rectangle
Set: [Optional] Rectangle | list | tuple		 The coordinates defining the reduced area where the routine runs. Point [0,0] corresponds to the upper left corner of the image, and point [1,1] to the lower right corner.
resolution [Optional] str Scanning resolution to be used during the routine.
