Data structures

Data structures are simple objects for transporting data. They may contain several auxiliary functions and/or overloaded operators for more convenience. Structures are often used to combine multiple function arguments (for example MoveSettings), or to represent data formats (AdornedImage).

AdornedImage Contains microscope image data along with metadata describing the state of the microscope under which the image was acquired.
AdornedImageMetadata Image metadata that captures the state of the microscope at the time of image acquisition.
AdornedImageMetadataAcquisition Section of image metadata related to image acquisition.
AdornedImageMetadataBinaryResult Section of image metadata that describes the properties of the resulting image.
AdornedImageMetadataCore Section of image metadata that describes software environment on the microscope computer.
AdornedImageMetadataDetector Section of image metadata related to the detector used for image acquisition.
AdornedImageMetadataEdsMap Section of image metadata describing the result of the EDS elemental or phase mapping experiment.
AdornedImageMetadataEnergyFilterSettings Section of image metadata describing properties of the electron or ion beam.
AdornedImageMetadataGasInjectionSystem Section of image metadata describing the state of the gas injection system.
AdornedImageMetadataGasInjectionSystemGas Section of image metadata describing one gas in the gas injection system.
AdornedImageMetadataInstrument Section of image metadata describing the properties of the microscope system and its software environment.
AdornedImageMetadataOptics Section of image metadata explaining the optical parameters of the electron or ion beam column.
AdornedImageMetadataOpticsAperture Section of image metadata related to beam apertures.
AdornedImageMetadataOpticsScanFieldSize Section of image metadata related to scan field size.
AdornedImageMetadataSample Section of image metadata describing the properties of the sample.
AdornedImageMetadataScanSettings Section of image metadata describing scan settings.
AdornedImageMetadataStageSettings Section of image metadata describing stage settings.
AdornedImageMetadataVacuumProperties Section of image metadata describing vacuum properties.
AdornedSpectrum Contains EDS spectrum data and metadata.
AdornedSpectrumMetadata Metadata describing one EDS spectrum.
AdornedSpectrumMetadataDetector Section of spectrum metadata that refers to the EDS detector used for spectrum acquisition.
AdornedSpectrumMetadataExperiment Section of spectrum metadata describing the experiment.
AdornedSpectrumMetadataRateStatistics Section of spectrum metadata with statistics on X-ray pulses.
AdornedSpectrumMetadataScene Section of spectrum metadata describing beam and stage properties.
AdornedSpectrumMetadataSpectrum Section of spectrum metadata describing details of the acquired spectrum.
AdornedSpectrumMetadataStatistics Section of spectrum metadata with statistics on X-ray pulses.
AdornedSpectrumMetadataSystem Section of spectrum metadata describing properties of the microscope system.
AdornedSpectrumMetadataTimeStatistics Section of spectrum metadata with additional time-related statistics.
BitmapPatternDefinition Contains the definition of a bitmap pattern.
CapProbeCalibration Defines the Cap Probe scale factor.
CapProbeStatistics Contains statistical data calculated from multiple Cap Probe measurements.
CapProbeTrainedValue Defines the Cap Probe set point for a stage tilt angle.
ChemicalCompositionEntry Holds quantification data of one chemical element.
ChemicalElement Represents one chemical element.
Color Represents the RGBA color model. The alpha channel controls the transparency of the color, while the other channels determine the color intensity. Values are in the range of [0, 1] corresponding to 0% - 100%.
ColorTable An auxiliary class providing a selection of named colors that align with the color scheme used in the main microscope user interface for rendering EDS elemental and phase maps.
CompustagePosition Defines a Compustage position.
DetectorInfo Contains information about the detector's current state and physical properties.
DetectorInsertSettings Contains additional settings for the function insert().
DetectorRetractSettings Contains additional settings for the function retract().
EdsElementalAnalysisResult Contains the results of elemental analysis of the EDS stream, obtained from the function run_elemental_analysis().
EdsElementalAnalysisSettings Contains settings for the EDS elemental analysis, which is conducted by the function run_elemental_analysis().
EdsPhaseAnalysisResult Holds the outcomes of phase analysis conducted on the EDS stream, retrieved from the function run_phase_analysis().
EdsPhaseAnalysisSettings Contains settings for the EDS phase analysis, which is conducted by the function run_phase_analysis().
EdsPhaseStatisticsEntry Contains statistical data for a single phase identified in an EDS stream. The function run_phase_analysis() returns a list of these structures for each phase it identifies in the input stream.
GetImageSettings Contains additional settings for the get_image() function.
GetRtmDataSettings Contains settings for the function get_data().
GetRtmPositionSettings Contains settings for the function get_positions().
GrabFrameSettings Contains additional settings for the grab_frame() function.
ImageMatch Contains the result of matching a template in an image.
KeyValue A generic structure that contains a key and value of type string.
Limits Defines maximum and minimum values of a scalar quantity.
Limits2d Defines maximum and minimum values of a 2D quantity.
ManipulatorPosition Defines a manipulator position.
MoveSettings Contains additional options for a stage move.
PeriodicTable An auxiliary class representing the periodic table of chemical elements.
Point Defines the X and Y coordinates of a point.
Rectangle Defines a rectangle by the coordinate of its upper-left corner, its width and height.
Region Specifies the area for image or spectrum acquisition.
RtmDataSet Defines a pattern point data set.
RtmPosition Defines a pattern point position.
RtmPositionSet Contains a set of pattern point positions.
RunAutoCbSettings Contains settings for the automatic brightness and contrast adjustment routine.
RunAutoFocusSettings Contains settings for the automatic focus routine.
RunAutoLensAlignmentSettings Contains settings for the automatic lens alignment adjustment routine.
RunAutoSourceTiltSettings Contains settings for the automatic source tilt adjustment routine.
RunAutoStigmatorCenteringSettings Contains settings for the automatic adjustment of stigmator center.
RunAutoStigmatorSettings Contains settings for the automatic stigmator adjustment routine.
StagePosition Defines a stage position.
StageVelocity Defines a stage velocity.
StreamPatternDefinition Contains the definition of a stream pattern.
TemperatureSettings Contains settings for controlling temperature of heating or cooling stages.
ToolReadinessAlignment Defines properties of an alignment in the ToolReadiness service.
ToolReadinessAlignmentParameter Specifies the name and value of a parameter for an alignment in the ToolReadiness service.
ToolReadinessAlignmentStatus Specifies the status of an alignment in the ToolReadiness service.
ToolReadinessDirectAlignment Defines properties of a direct alignment in the ToolReadiness service.
ToolReadinessDirectAlignmentStatus Specifies the status of a direct alignment in the ToolReadiness service.
ToolReadinessUseCaseSettings Specifies settings for the usecase of a ToolReadiness alignment.
VacuumSettings Contains settings for the function pump().
