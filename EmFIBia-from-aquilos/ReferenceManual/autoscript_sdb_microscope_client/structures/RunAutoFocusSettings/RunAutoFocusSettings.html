RunAutoFocusSettings

Contains settings for the automatic focus routine.

Items

method [Optional] str The name of the auto focus algorithm variant.
dwell_time [Optional] float The scanning dwell time to apply during the routine.
horizontal_field_width [Optional] float The value of horizontal field width to use during the routine.
line_integration [Optional] int The number of lines to integrate when acquiring image in each auto function iteration.
number_of_frames [Optional] int The number of frames to integrate when acquiring image in each auto function iteration.
reduced_area Get: [Optional] Rectangle
Set: [Optional] Rectangle | list | tuple		 The coordinates defining the reduced area where the autofocus routine runs. Point [0,0] corresponds to the upper left corner of the image, and point [1,1] to the lower right corner.
resolution [Optional] str The scanning resolution to be used during the routine.
working_distance_step [Optional] float The working distance step to be used by the routine while searching for the optimal focus setting.
display_point Get: [Optional] Point
Set: [Optional] Point | list | tuple		 The center point of the reduced area where the autofocus routine runs. For algorithm variants that permit this setting, the area size remains constant. Point [0,0] corresponds to the upper left corner of the view, and point [1,1] corresponds to the lower right corner.
maximum_iterations [Optional] int The maximum number of iterations the algorithm will perform to find the optimal focus setting. If the optimum is not found within this limit, the function will fail.
