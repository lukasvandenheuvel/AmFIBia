ImageMatch

Contains the result of matching a template in an image.

Items

center Get: Point
Set: Point | list | tuple		 The Point structure containing coordinates of the center of the match in the original image, in pixels.
score [Optional] float The confidence level with which the template matching algorithm found the template, in the range of [0, 1] corresponding to 0% - 100%.

Remarks

This structure contains the result of a template matching routine such as match_template().
