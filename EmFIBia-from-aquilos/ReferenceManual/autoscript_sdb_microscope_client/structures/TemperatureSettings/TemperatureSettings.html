TemperatureSettings

Contains settings for controlling temperature of heating or cooling stages.

Items

target_temperature [Optional] float The target temperature of the stage. The default unit is Kelvin, but you can configure a preferred unit by calling set_preferred_unit().
ramping_speed [Optional] float The temperature ramping speed. The default unit is Kelvin per second, but you can configure a preferred unit by calling set_preferred_unit().
soak_time [Optional] float Time in seconds to remain at the target temperature after it has been reached within the specified tolerance.
tolerance [Optional] float Minimum desired difference between target and actual temperatures. The default unit is Kelvin, but you can configure a preferred unit by calling set_preferred_unit().
timeout [Optional] int Maximum time the temperature ramping can last, in seconds.
