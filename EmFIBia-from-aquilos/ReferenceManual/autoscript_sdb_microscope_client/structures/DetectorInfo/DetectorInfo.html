DetectorInfo

Contains information about the detector's current state and physical properties.

Items

type [Optional] str The type of detector. One of the values from the DetectorType enumeration.
state [Optional] str The current insertion state of a retractable detector. One of the values from the RetractableDeviceState enumeration.
is_retractable [Optional] bool Returns true if the detector is retractable, meaning it must be inserted into the specimen chamber before image acquisition can occur.
can_insert [Optional] bool Returns true if the detector can be inserted given the current microscope state.
can_retract [Optional] bool Returns true if the detector can be retracted given the current microscope state.
is_enabled [Optional] bool Returns true if the detector can be activated in the current view based on the current microscope state.
