DetectorInfo.is_enabled

Returns true if the detector can be activated in the current view based on the current microscope state.

Data type

[Optional] bool

Remarks

This property primarily indicates whether the detector is compatible with the imaging device configured for the active view and if the detector hardware is ready for image acquisition. If the detector is retractable, it can be activated for a view even when retracted (i.e., is_enabled will be true).
