MoveSettings

Contains additional options for a stage move.

Items

rotate_compucentric [Optional] bool The option specifies if compucentric rotation should be used in the move.
link_z_y [Optional] bool The option specifies if z-y compensation should be used in the move.
link_z_b [Optional] bool The option specifies if z-beta compensation should be used in the move. Supported for Compustage movements only.
z_cap_move [Optional] bool The option specifies if the Cap Probe sensor should be used in the move to adjust the Z axis.
tilt_compucentric [Optional] bool The option specifies if compucentric tilt should be used in the move.
use_coarse_only [Optional] bool When set to True, the stage is moved only with the coarse motors. Applicable only on systems equipped with the piezo stage.
use_fine_only [Optional] bool When set to True, the stage is moved only with the fine motors. Applicable only on systems equipped with the piezo stage.

Remarks

The structure is used with the functions specimen.stage.absolute_move(), specimen.stage.relative_move() and their equivalents on the compustage object - specimen.compustage.absolute_move() and specimen.compustage.relative_move().

For stage moves: If any of the move settings items are not initialized, they default to the values set by the function specimen.stage.set_default_move_settings() or to False if the function was not yet used to set the default values.

For compustage moves: If any of the move settings items are not initialized, they default to False.
