DetectorInsertSettings

Contains additional settings for the function insert().

Items

insert_position [Optional] str The detector insert position. You can use the Stem4Positions or EdsDetectorPositions enumeration to specify this item.
type [Optional] str The type of the detector to be inserted. The DetectorType enumeration lists all supported detector types.

Remarks

The insert_position is required for both the STEM4 detector and the EDS detector.
