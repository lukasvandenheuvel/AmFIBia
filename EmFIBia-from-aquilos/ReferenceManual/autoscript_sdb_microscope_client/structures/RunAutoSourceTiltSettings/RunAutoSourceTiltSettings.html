RunAutoSourceTiltSettings

Contains settings for the automatic source tilt adjustment routine.

Items

method [Optional] str The name of the auto source tilt algorithm variant.
contrast [Optional] float The detector contrast to be used during the routine.
brightness [Optional] float The detector brightness to be used during the routine.
dwell_time [Optional] float The scanning dwell time to apply during the routine.
resolution [Optional] str The resolution to be used during the routine.
