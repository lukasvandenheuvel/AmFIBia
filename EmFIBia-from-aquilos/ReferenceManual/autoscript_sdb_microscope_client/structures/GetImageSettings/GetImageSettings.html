GetImageSettings

Contains additional settings for the get_image() function.

Items

wait_for_frame [Optional] bool If True, the get_image() function blocks until acquisition reaches the end of the frame and then returns the image.
view [Optional] int The 1-based index of the view from which the image will be copied. If not specified, the active view is used.
timeout [Optional] float The maximum wait time, in seconds, for the completion of one frame. Applicable only when wait_for_frame is set to True.
