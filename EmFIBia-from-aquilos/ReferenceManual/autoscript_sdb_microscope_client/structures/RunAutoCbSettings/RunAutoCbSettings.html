RunAutoCbSettings

Contains settings for the automatic brightness and contrast adjustment routine.

Items

black_target [Optional] float The grayscale level at which the black pixels should be placed. The value is expected to be in the range of [0, 1] corresponding to 0% - 100%.
white_target [Optional] float The grayscale level at which the white pixels should be placed. The value is expected to be in the range of [0, 1] corresponding to 0% - 100%.
method [Optional] str The name of the auto contrast brightness algorithm variant.
calibrate_detector [Optional] bool Whether to force detector calibration prior to running the routine. Normally, it is sufficient to calibrate the detector only once. Using this option, you can make the routine more robust in exchange for a slightly longer duration.
dwell_time [Optional] float The scanning dwell time to apply during the routine.
line_integration [Optional] int The number of lines to integrate when acquiring image in each auto function iteration.
max_black_clipping [Optional] float The maximum percentage of black pixels which can be clipped during the routine. The value is expected to be in the range of [0, 1] corresponding to 0% - 100%.
max_white_clipping [Optional] float The maximum percentage of white pixels which can be clipped during the routine. The value is expected to be in the range of [0, 1] corresponding to 0% - 100%.
number_of_frames [Optional] int The number of frames to integrate when acquiring image in each auto function iteration.
resolution [Optional] str The scanning resolution to be used during the routine.
brightness_target [Optional] float The brightness level at which the median brightness pixels should be placed. The value is expected to be in the range of [0, 1] corresponding to 0% - 100%.
