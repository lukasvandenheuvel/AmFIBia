GrabFrameSettings.smart_frame_integration

If true, the smart frame integration technique is employed during image acquisition to reduce noise and blur. The parameters dwell_time and frame_integration must be specified along with this parameter.

Data type

[Optional] bool

Remarks

  • Smart Frame Integration is an online method and requires interaction with the microscope. It analyzes the frames as they are being scanned and dynamically changes detector settings so that the best possible image is obtained. Images taken at different detector settings are fused together with an HDR-like algorithm. In the background, it also analyzes various image quality metrics such as the noise and blur present in the image. It uses the metrics to set optimal parameters for digital denoising, deconvolution, and local contrast enhancement algorithms. Finally, it uses the noise estimates to automatically terminate the frame integration after enough information about the sample has been collected.
  • Smart frame integration is currently supported only on Apreo ChemiSEM system (software platform version 37.x).
  • Some acquisition parameters are incompatible with the smart frame integration technique, including: line_integration, scan_interlacing and reduced_area. If incorrect setting combinations are passed to imaging.grab_frame(), an exception will be raised with a hint message indicating what needs to be adjusted.
