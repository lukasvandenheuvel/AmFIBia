AdornedSpectrum

Contains EDS spectrum data and metadata.

Items

total_counts int The total number of characteristic X-ray counts in the spectrum.
data [Read only] Numpy.ndarray Spectrum data arranged in 1D array. The element at index i represents the x-ray count at energy level i * BinSize eV. The value of BinSize can be read from spectrum metadata.
metadata [Optional] AdornedSpectrumMetadata Metadata describing the EDS spectrum.

Methods

load() Loads a spectrum from an EMSA file.
save() Saves the spectrum at the specified path as an EMSA file.

Remarks

You can add two spectra that have been acquired by the same detector and have same data size. The result of this operation is a summarized dataset with combined metadata values. The date and time are set to the actual time. Adding spectra is commutative.

Examples

Acquire spectra from the left and right halves of the field of view, then add them together: 

        region1 = Region(type=RegionType.RECTANGLE, rectangle=[0, 0, 0.5, 1], coordinate_system=RegionCoordinateSystem.REDUCED_AREA)
        spectrum1 = microscope.analysis.eds.acquire_spectrum(region1, 5)
        print("Spectrum 1 total counts:", spectrum1.total_counts)
        
        region2 = Region(type=RegionType.RECTANGLE, rectangle=[0.5, 0, 0.5, 1], coordinate_system=RegionCoordinateSystem.REDUCED_AREA)
        spectrum2 = microscope.analysis.eds.acquire_spectrum(region2, 5)
        print("Spectrum 2 total counts:", spectrum2.total_counts)
        
        sum_spectrum = spectrum1 + spectrum2
        print("Sum spectrum total counts:", sum_spectrum.total_counts)
