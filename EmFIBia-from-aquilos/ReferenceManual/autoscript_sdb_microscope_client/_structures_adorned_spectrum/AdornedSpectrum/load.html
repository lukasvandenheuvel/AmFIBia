AdornedSpectrum.load

Loads a spectrum from an EMSA file.

Syntax

load(path)

Parameters

path The path to the EMSA file from which to load the spectrum.

Return value

AdornedSpectrum The loaded spectrum as an AdornedSpectrum object.

Remarks

The EMSA format is standardized by ISO 22029:2022. However, AutoScript supports only EMSA files with specific attributes commonly used in EDS analysis applications. The following list outlines the required values of selected EMSA attributes. Files that do not meet these criteria will not be loaded.
  • #SIGNALTYPE = "EDS"
  • #XUNITS = "eV"
  • #YUNITS = "counts"
  • #DATATYPE = "XY"
  • #NCOLUMNS = 1

Examples


        # Load spectrum from the given path
        spectrum = AdornedSpectrum.load(r"C:\spectrum.emsa")
