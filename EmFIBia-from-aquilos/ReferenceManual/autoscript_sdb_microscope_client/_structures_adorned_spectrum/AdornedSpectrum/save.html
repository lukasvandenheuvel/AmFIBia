AdornedSpectrum.save

Saves the spectrum at the specified path as an EMSA file.

Syntax

save(path)

Parameters

path The location where the spectrum will be saved.

Remarks

The spectrum object on which save() is called is a client-side object. Therefore, the path parameter specifies a location on the client computer, which is typically different from the microscope PC.

Examples


        # Acquire spectrum from the entire field of view
        region = Region(type=RegionType.FULL_FRAME)
        spectrum = microscope.analysis.eds.acquire_spectrum(region, 3)
        
        # Save the spectrum data and related metadata to EMSA file
        spectrum.save(r"c:\spectrum.emsa")
