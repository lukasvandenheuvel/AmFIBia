Enumerations

Enumeration types do not appear directly in the API as function parameters or return values. Instead, the API uses simple types like strings or integers. However, using enum values rather than hard-coded constants is recommended to minimize typos and improve code readability.
If there is a possibility to supply an enumeration value for a function parameter, this manual will mention it in the function description.

AlignmentMillingProcess Type of milling process used by some of the alignments from the ToolReadiness service.
AngularCorrectionMode Specifies the supported angular correction modes.
AutoFunctionMethod An enumeration of all auto function methods.
AutoLensAlignmentModulationType Specifies the modulation types for the automatic lens alignment adjustment routine.
AutoloaderCompartmentType Specifies the Autoloader device compartment type.
BeamType Specifies the supported beam types.
CameraEmissionType The color of light emitted by the LED used for sample illumination in fluorescence light microscope camera (FLM) acquisitions.
CameraFilterType The type of optical filter used in fluorescence light microscope camera (FLM) acquisitions.
ChemicalCompositionLineFamily Specifies selected line family in the ChemicalCompositionEntry.
CoordinateSystem Specifies the coordinate systems in which stage coordinates can be given.
CryoCompartmentTemperature Specifies the cryo compartment temperature presets.
DetectorMode Specifies the supported detector modes.
DetectorSegment Specifies the names of detector segments.
DetectorType Specifies the supported detector types.
EdsDetectorPositions Specifies the available EDS detector positions.
EdsMappingEngineType Specifies the possible types of mapping engines for generating EDS elemental maps.
EdsSegmentationFilterMask Defines different image filter strengths for image segmentation within the EDS phase mapping engine. The two 'Auto' options are designed for dynamic segmentation resolutions.
EdsSegmentationFilterType Defines different filter settings for reducing the noise in the input data within the EDS phase mapping engine.
EdsSegmentationMethod The type of image segmentation algorithm for the EDS elemental mapping engine.
EdsSegmentationResolution Represents different resolution settings for image segmentation in the EDS phase mapping engine. It includes options for dynamic resolution, which applies varying resolutions to individual segmentation layers, and preset resolutions, which maintain a fixed resolution across all layers.
ElectronBeamMode Specifies the supported modes of the electron beam source.
ImageDataEncoding Specifies the supported image encodings.
ImageFileFormat Specifies the supported image file formats.
ImagingDevice Specifies the supported imaging devices.
ImagingState Specifies the possible states of image acquisition.
LoadLockState Specifies the possible states of the LoadLock device.
Location Specifies location in space.
ManipulatorCoordinateSystem Specifies the coordinate systems in which manipulator (EasyLift) coordinates can be given.
ManipulatorSavedPosition Specifies the names of saved manipulator positions which can be retrieved by the function get_saved_position().
ManipulatorState Specifies the insertion states of the manipulator needle.
MultiChemGasState Defines the possible states of the MultiChem gas.
MultiChemInjectorState Defines possible states of the MultiChem injector.
MultiChemInsertPosition Specifies the standard insert positions of the MultiChem device.
OpticalMode Specifies the SEM optical modes.
Orientation Specifies orientation in space.
PatterningMode Specifies the modes of the patterning process.
PatterningState Specifies the possible states of the patterning process.
PatternScanDirection Specifies the possible directions of scanning for a pattern.
PatternScanType Specifies the scanning strategies for a pattern.
PlasmaGasType Specifies gases that can be set to the Multiple Ion Plasma FIB source.
ProtectiveShutterMode Specifies the available modes of the protective shutter.
Quantity Defines the physical quantities for which the preferred unit can be configured instead of the default SI unit. Future versions of AutoScript are planned to include additional quantities in this enumeration.
RegionCoordinateSystem Defines the coordinate system for a region used for spectrum acquisition.
RegionType Defines the type of region for image or spectrum acquisition.
RegularCrossSectionScanMethod Specifies the supported scan methods for a regular cross section pattern.
RetractableDeviceState Specifies the possible states of a retractable device.
RtmCoordinateSystem Specifies the available RTM coordinate systems.
RtmMode Specifies the available RTM resolution modes.
SampleMapType Specifies sample map variants.
ScanningFilterType Specifies the supported types of the scanning filter.
ScanningMode Specifies the available scanning modes for electron or ion beams.
ScanningResolution Specifies the standard scan resolutions.
SegmentPolarity Specifies the polarity of a detector segment.
StageAxis An enumeration of stage axes.
Stem4Positions Specifies the available STEM4 positions.
StopAcquisitionMethod Specifies how the acquisition stop operation should be performed.
TemperatureStageHeatShieldType Specifies the types of heat shields for a heating stage.
TemperatureStageType Specifies the types of temperature stages.
ToolReadinessAlignmentName Alignment names known to the ToolReadiness service.
ToolReadinessAlignmentState Represents the status of the alignment in the ToolReadiness service.
Unit An enumeration of units that can be set as preferred for a physical quantity.
VacuumGasType Specifies the types of gases for the ESEM vacuum mode.
VacuumMode Specifies the vacuum modes of the specimen chamber.
VacuumState Specifies the possible states of specimen chamber vacuum.
