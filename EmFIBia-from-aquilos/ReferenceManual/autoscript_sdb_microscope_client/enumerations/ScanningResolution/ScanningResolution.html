ScanningResolution

Specifies the standard scan resolutions.

Values

PRESET_1024X884 "1024x884"
PRESET_1536X1024 "1536x1024"
PRESET_2048X1768 "2048x1768"
PRESET_3072X2048 "3072x2048"
PRESET_4096X3536 "4096x3536"
PRESET_512X442 "512x442"
PRESET_6144X4096 "6144x4096"
PRESET_768X512 "768x512"

Remarks

  • This enumeration can be passed as a function parameter where a scan resolution is expected in the format "WIDTHxHEIGHT".
  • Image acquisition in resolutions from this enumeration result in more efficient "standard" imaging technique, in contrast to the "large image acquisition" technique, which works with any resolutions up to 65536x65536, but is less efficient and limited in applicable scan settings. For more information, see User guide: Imaging and User guide: Large image acquisition.
