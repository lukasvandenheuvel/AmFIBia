ScanningFilterType

Specifies the supported types of the scanning filter.

Values

AVERAGING 2 Continuously averages a specified number of frames (2 or more), resulting in a better signal-to-noise ratio. During averaging, the image is updated continuously and actions such as focusing and moving the stage can still be performed. This process will continue until stopped by change of scanning condition or by freezing the result.
INTEGRATION 3 Allows accumulative noise reduction by true integration over a number of frames and freezes the final image. This process continues until the predefined number of frames is reached, and then stops and freezes automatically.
NONE 1 Live imaging with one frame following the other. The image remains unfiltered for collecting direct images, mostly in Live/Slow scan. This is the raw scanned image with no filtering.

Remarks

The enumeration is meant to be used with imaging.scanning_filter.type. To set the number of frames for averaging or integration, use the number_of_frames property.
