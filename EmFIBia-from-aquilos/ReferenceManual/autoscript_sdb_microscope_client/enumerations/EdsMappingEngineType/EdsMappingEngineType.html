EdsMappingEngineType

Specifies the possible types of mapping engines for generating EDS elemental maps.

Values

GROSS_COUNT "GrossCount" A basic elemental mapping engine that marks the presence of a chemical element in a pixel by altering the pixel's color. The only segmentation method applicable for this engine type is NONE.
PHASE "Phase" A phase mapping engine, which targets identifying unique phases (pixels of the same composition).
QUANT "Quant" A sophisticated elemental mapping engine that correlates chemical information acquired from the EDS detector with sample topology information obtained from a standard SEM detector. The applicable segmentation methods for this engine are REGULAR and OBJECT_BASED.
