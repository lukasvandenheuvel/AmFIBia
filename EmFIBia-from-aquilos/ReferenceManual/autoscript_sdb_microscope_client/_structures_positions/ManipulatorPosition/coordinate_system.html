ManipulatorPosition.coordinate_system

The coordinate system in which the position is specified. You can use the ManipulatorCoordinateSystem enumeration to select the coordinate system.

Data type

[Optional] str

Examples


        # Move manipulator in Stage coordinate system
        position = ManipulatorPosition(x=0.0001, y=0.0001, coordinate_system=ManipulatorCoordinateSystem.STAGE)
        microscope.specimen.manipulator.absolute_move(position)
        
        # Move manipulator in Raw coordinate system
        position = ManipulatorPosition(x=0.0001, y=0.0001, coordinate_system=ManipulatorCoordinateSystem.RAW)
        microscope.specimen.manipulator.absolute_move(position)
