StagePosition

Defines a stage position.

Items

x [Optional] float X coordinate.
y [Optional] float Y coordinate.
z [Optional] float Z coordinate.
r [Optional] float Rotation.
t [Optional] float Tilt.
coordinate_system [Optional] str The coordinate system in which the position is specified. You can use the CoordinateSystem enumeration to select the coordinate system.

Remarks

  • Property specimen.stage.current_position returns an object of this type to represent the current stage position.
  • Stage manipulation functions specimen.stage.relative_move() and specimen.stage.absolute_move() take a parameter of this type to specify stage movement distance.
