Microscope API

The autoscript_sdb_microscope_client Python package provides functionalities for controlling microscope systems from Thermo Fisher Scientific, specifically SEM (scanning electron microscope) and DualBeam families.

The ThermoScientific.ASAPI NuGet package includes assemblies that encapsulate interfaces, classes, and data types required for controlling Thermo Fisher Scientific SEM systems.

The ProSEM API C++ SDK package offers headers and static libraries that define and implement the interfaces, classes, and data types required for controlling Thermo Fisher Scientific SEM systems.

The primary class in this package is SdbMicroscopeClient. Upon instantiation and connection to an AutoScript server, this class offers a range of functions through its members, organized in a static tree structure. These members include child objects, properties, and functions that facilitate various control operations.

The package also includes other classes that enhance its capabilities. Data structures are designed to provide comprehensive parameter definitions and return values for functions within the static tree. Enumerations are employed to minimize potential typos by using predefined values instead of raw strings or integers. Dynamic objects, on the other hand, have a shorter lifespan compared to SdbMicroscopeClient and are typically created explicitly during script execution.
