patterning

Enables control and provides the status of the microscope patterning engine.

Classes

ispi Enables control over the intermittent simultaneous patterning and imaging (iSPI) process.
real_time_monitor Enables control and provides the status of the Real Time Monitor (RTM).

Properties

mode Gets and sets the active patterning mode.
state [Read only] Gets the current state of the patterning engine.

Methods

clear_patterns() Deletes all existing patterns in the active view.
create_bitmap() Creates a bitmap pattern.
create_circle() Creates a circular pattern.
create_cleaning_cross_section() Creates a cleaning cross section pattern.
create_line() Creates a line pattern.
create_polygon() Creates a polygon pattern.
create_rectangle() Creates a rectangular pattern.
create_regular_cross_section() Creates a regular cross section pattern.
create_stream() Creates a stream file pattern.
get_patterns() Returns a list of patterns created in the active view for the current imaging device.
list_all_application_files() Returns all application files available on the system.
pause() Pauses a patterning job in progress.
resume() Resumes a paused patterning job.
run() Executes a patterning job synchronously, i.e. waits for the patterning to end.
set_default_application_file() Sets the default application file to be assigned to newly created patterns.
set_default_beam_type() Sets the beam type to be assigned to newly created patterns.
start() Starts a patterning job asynchronously.
stop() Aborts a patterning job in progress.
