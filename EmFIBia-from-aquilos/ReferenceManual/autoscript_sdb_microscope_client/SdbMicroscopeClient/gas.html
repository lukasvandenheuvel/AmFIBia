gas

Enables control of the gas injection system (GIS).

Properties

is_multichem_pid_enabled Switches MultiChem PID gas flow control on/off.

Methods

get_gis_port() Returns a GIS port with the specified gas.
get_multichem() Returns an object representing the MultiChem device located at the specified port.
list_all_gis_ports() Returns a list of gases for all GIS ports installed on the system.
list_all_multichem_ports() Returns a list of MultiChem devices installed on the system. Each device is described by a port identifier.
