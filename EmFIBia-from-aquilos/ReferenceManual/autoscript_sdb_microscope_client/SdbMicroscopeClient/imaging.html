imaging

Enables control of the microscope imaging.

Classes

lut Contains functions to control digital contrast and brightness in the active view.
scanning_filter Enables control of the microscope scanning filter.

Properties

state [Read only] Gets the current state of the imaging.

Methods

get_active_device() Returns the type of imaging device set in the active view.
get_active_view() Returns the index of the active view.
get_available_devices() Returns a list of imaging devices installed on the microscope.
get_image() Retrieves a microscope image currently present in the active view.
grab_frame() Scans a new frame in the active view and returns the acquired image.
grab_frame_to_disk() Scans a completely new frame in the active view, saves the image to disk and provides the preview image.
grab_multiple_frames() Scans new frames in all compatible views and returns the acquired images.
match_template() Tries to locate a feature from the given template image in the given searched image.
set_active_device() Changes the active imaging device in the active view.
set_active_view() Changes the active view in the main user interface of the microscope (application 'Microscope Control').
set_image() Uploads an image into the active view, including its metadata.
start_acquisition() Starts live image acquisition with the active device in the active view.
stop_acquisition() Stops live image acquisition in the active view.
