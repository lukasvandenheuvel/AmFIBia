detector

Enables control and provides the status of the active detector, i.e. the detector currently set in the active view.

Classes

brightness Enables control of the voltage offset for the active detector.
camera_settings Contains settings of several camera detectors - Optical microscope (HiResOptical), Fluorescence light microscope (FLM) and others.
contrast Enables control of the electronic gain for the active detector.
custom_settings Additional detectors settings.
mode Enables control of the active detector mode.
type Enables control of the active detector type.

Properties

is_homed [Read only] Returns True if the active detector is homed. The homing procedure is only applicable for some of the camera detectors, e.g. for the fluorescence light microscope.
state [Read only] Returns the current state of a retractable detector. Possible values are defined in the RetractableDeviceState enumeration.

Methods

get_info() Provides details on the detector's physical properties and current state.
home() Homes the active detector. The homing procedure is only applicable for some of the camera detectors, e.g. for the fluorescence light microscope.
insert() Inserts the active detector unless a specific detector type is provided in settings.
retract() Retracts the active detector unless a specific detector type is provided in settings.
set_type_mode() Sets both the active detector type and mode.

Remarks

Active detector can be set using property detector.type.value or method set_type_mode().
