SdbMicroscopeClient

Methods

connect() This method connects the AutoScript client to the AutoScript server.
disconnect() This method disconnects the AutoScript client from the AutoScript server.

Properties

server_host [Read only] Returns the hostname of the AutoScript server to which the AutoScript client is currently connected.

Remarks

This is the top-level class of the autoscript_sdb_microscope_client package. It provides a wide range of functionality through its members, which are organized in a static tree. This page only contains information about methods related to server connectivity. The microscope API organized in a static tree is documented separately.
