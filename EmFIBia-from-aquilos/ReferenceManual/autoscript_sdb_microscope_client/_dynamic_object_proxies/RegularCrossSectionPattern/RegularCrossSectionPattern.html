RegularCrossSectionPattern

The regular cross section pattern type.

Properties

application_file The name of the application file.
beam_type The beam used for patterning, either electron or ion.
blur Like Defocus, but specifying the additional diameter of the blurred spot.
center_x The X coordinate of the pattern center position.
center_y The Y coordinate of the pattern center position.
defocus The defocus (WD change) of the beam. It allows focusing above (negative value) or below (positive value) the sample surface.
depth The depth of the pattern.
dose Defines the charge dose per area in C/µm².
dwell_time The time the beam spends on a single pixel per pass (rounded to a multiple of 25 ns).
enabled If set to False, the pattern is removed from the patterning job.
gas_flow Sets the custom gas flow for the pattern. Make sure to set the gas type first before setting the gas flow.
gas_needle_position The position of the MultiChem needle when patterning begins.
gas_type The name of the gas required to pattern this shape, or an empty string if no gas should be used.
height The pattern height.
id [Read only] A unique pattern identifier.
interaction_diameter The interaction diameter for an infinitely small beam in meters.
is_exclusion_zone If set to True, the pattern area is not to be processed.
multi_scan_pass_count The number of passes (scans) to be performed over the pattern area for the multi-scan method.
overlap_x Sets the beam diameter overlap in X direction.
overlap_y Sets the beam diameter overlap in Y direction.
pass_count The number of passes (scans) to be performed over the pattern area.
pitch_x Sets the pitch X between two spots.
pitch_y Sets the pitch Y between two spots.
refresh_time The minimum loop time that must at least elapse before the next pass, so that the adsorbed gas can be refreshed.
rotation The pattern rotation angle.
scan_direction The direction of the scan movement.
scan_method The scanning method used while patterning. You can use RegularCrossSectionScanMethod enumeration to specify the value.
scan_ratio Gets or sets the ratio between the dose delivered at the first line and the dose delivered at the last line of the pattern.
scan_type The scanning strategy used while patterning.
time The time required to process the pattern.
volume_per_dose The volume of material that is removed per charge in mm³/nC.
width The pattern width.
