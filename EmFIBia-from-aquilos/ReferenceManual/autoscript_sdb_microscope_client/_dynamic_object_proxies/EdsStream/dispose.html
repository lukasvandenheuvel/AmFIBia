EdsStream.dispose

Frees all resources allocated on the AutoScript server side for this EDS stream. Once the object is disposed, performing operations on the EDS data it represents is no longer possible.

Syntax

dispose()

Remarks

Elemental and phase analysis of EDS streams can place significant demands on the memory and CPU resources of the microscope computer. If you are working with multiple stream objects, it is recommended to call the dispose() function as soon as all processing is complete to release allocated resources even before they are automatically released when the client session is terminated.
