EdsStream.get_scene_spectrum

Returns the scene (full-frame) spectrum calculated from the EDS stream. The stream must be closed before calling this function.

Syntax

get_scene_spectrum()

Return value

AdornedSpectrum An AdornedSpectrum object containing the scene spectrum data along with its metadata.

Examples


        # Acquire EDS data by scanning (not shown here) 
        # ... 
        
        # Get the active EDS stream and close it to enable processing 
        eds_stream = microscope.analysis.eds.mapping.get_stream()
        eds_stream.close()
        
        # Get the scene (full-frame) spectrum and save it 
        scene_spectrum = eds_stream.get_scene_spectrum()
        scene_spectrum.save("c:\\scene_spectrum.emsa")
