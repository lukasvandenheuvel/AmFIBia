EdsStream.run_elemental_analysis

Analyzes the data contained within the EDS stream and prepares a mapping engine for elemental map generation. The stream must be closed before calling this function.

Syntax

run_elemental_analysis()
run_elemental_analysis(settings)

Parameters

settings EdsElementalAnalysisSettings Parameters of the elemental analysis, primarily specifying properties of the mapping engine.

Return value

EdsElementalAnalysisResult The analysis result, which includes both a list of elements identified in the EDS stream and the calculated chemical composition.

Remarks

  • The Quant mapping engine type and object-based segmentation method are the default configuration of the mapping algorithm unless overridden in the settings parameter.
  • Conducting an analysis is a mandatory step before calling the EdsStream.get_elemental_map() function to generate map images.
  • Only one mapping engine can be initialized at a time. If you want to generate maps with multiple engines, do it in succession. For example, first perform the analysis with the Quant engine, generate Quant maps, then repeat the analysis with the GrossCount engine and generate GrossCount maps.
