EdsStream

Represents an EDS data container, collecting EDS data as the electron beam scans. The data includes both the count of characteristic X-rays and their positions on the sample.

Properties

is_open [Read only] Returns True if the stream is presently collecting X-rays during electron beam scanning. To carry out data processing operations with the stream, it is necessary to close it first.
total_counts [Read only] Returns the total number of characteristic X-ray counts in the EDS stream.

Methods

close() Closes the EDS stream, indicating the end of the EDS experiment and allowing for the subsequent processing of the data stored in the stream.
dispose() Frees all resources allocated on the AutoScript server side for this EDS stream. Once the object is disposed, performing operations on the EDS data it represents is no longer possible.
get_elemental_map() Generates an elemental map image from the EDS stream for specified chemical elements. Before calling this function, it's necessary to prepare the mapping engine by executing run_elemental_analysis().
get_peak_id() Identifies the presence of chemical elements in the EDS stream. The stream must be closed before calling this function.
get_phase_map() The function generates either a single phase map or a composite map of several phases. Before calling this function, it's necessary to prepare the mapping engine by executing run_phase_analysis().
get_phase_spectrum() Provides an EDS spectrum collected from the area of the sample corresponding to the specified phase. Before calling this function, it's necessary to complete the phase analysis of the stream by executing run_phase_analysis().
get_scene_spectrum() Returns the scene (full-frame) spectrum calculated from the EDS stream. The stream must be closed before calling this function.
get_sem_image() Returns the SEM image that captures the scene at the moment the EDS stream was closed. This image, stored within the EdsStream object, serves a role in mapping applications. Mapping engines leverage the topological data extracted from this image to refine the distinction between chemically coherent regions, thereby enhancing boundary delineation accuracy.
run_elemental_analysis() Analyzes the data contained within the EDS stream and prepares a mapping engine for elemental map generation. The stream must be closed before calling this function.
run_phase_analysis() Performs an analysis of the EDS stream to identify distinct chemical phases within the sample. Also prepares the phase mapping engine, allowing it to generate phase maps upon subsequent calls to get_phase_map(). The stream must be closed before calling this function.
save() Exports the EDS stream into a file in Thermo Scientific PAD format. The stream must be closed before calling this function.
