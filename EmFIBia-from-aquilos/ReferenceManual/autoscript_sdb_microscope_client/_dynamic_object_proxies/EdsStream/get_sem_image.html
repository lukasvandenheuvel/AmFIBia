EdsStream.get_sem_image

Returns the SEM image that captures the scene at the moment the EDS stream was closed. This image, stored within the EdsStream object, serves a role in mapping applications. Mapping engines leverage the topological data extracted from this image to refine the distinction between chemically coherent regions, thereby enhancing boundary delineation accuracy.

Syntax

get_sem_image()

Return value

AdornedImage The SEM image as AdornedImage object.

Remarks

  • The returned image represents the "raw" data acquired by the active detector in the active view. It has not undergone any digital processing such as frame averaging, contrast enhancement etc. The image always has a bit depth of 8.
  • The blending of SEM images with generated elemental or phase maps is a common operation facilitated by the blend_images() function in the AutoScript Vision Toolkit.

Examples


        # Acquire EDS data by scanning (not shown here) 
        # ... 
        
        # Get the active EDS stream and close it to enable processing 
        eds_stream = microscope.analysis.eds.mapping.get_stream()
        eds_stream.close()
        
        # Get the raw SEM image associated with the EDS stream and save it
        sem_image = eds_stream.get_sem_image()
        sem_image.save("c:\\sem_image.tif")
