EdsStream.close

Closes the EDS stream, indicating the end of the EDS experiment and allowing for the subsequent processing of the data stored in the stream.

Syntax

close()

Remarks

  • Closing the stream triggers several actions:
    • Electron beam scanning is stopped to enable export of stream data.
    • All data contained in the stream is set aside for future processing. The copy is exclusively owned by AutoScript, so no further actions on the SEM can affect it.
    • A new, empty EDS stream is established in the platform software to register incoming X-rays.
    • AutoScript stores a single SEM image from the active view. The image is later analyzed by the mapping engine to identify sample segments and their boundaries.
  • It is safe to call close() even though the stream is already closed.
