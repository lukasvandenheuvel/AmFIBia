EdsStream.save

Exports the EDS stream into a file in Thermo Scientific PAD format. The stream must be closed before calling this function.

Syntax

save(path)

Parameters

path str The target PAD file path.

Remarks

The PAD format is a proprietary format developed by Thermo Fisher Scientific. The PAD file stores all EDS data and metadata from the EdsStream, including an 8-bit grayscale SEM image of the complete scene.
In the current version of AutoScript, it is not possible to save the generated color maps or mapping engine configurations, although the format allows for it.
The generated PAD file can be imported into the Microscope Control or ChemiView software from Thermo Fisher Scientific for further processing.

Examples


        # Assuming the active view is configured with the electron beam, scanning stopped
        
        # Create a new empty EDS stream and activate it 
        microscope.analysis.eds.mapping.reset_stream()
        
        # Scan the entire field of view for 10 seconds; the EDS stream will register X-rays generated by the scanning
        microscope.beams.electron_beam.scanning.mode.set_full_frame()
        microscope.imaging.start_acquisition()
        time.sleep(10)
        microscope.imaging.stop_acquisition()
        
        # Get the active stream and close it to enable processing
        eds_stream = microscope.analysis.eds.mapping.get_stream()
        eds_stream.close()
        
        # Save all data from the stream to disk in PAD format 
        eds_stream.save("experiment.pad")
