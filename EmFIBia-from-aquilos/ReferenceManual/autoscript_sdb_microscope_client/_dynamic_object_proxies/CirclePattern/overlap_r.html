CirclePattern.overlap_r

Sets the beam diameter overlap.

Data type

float

Remarks

  • The overlap value is specified as percentage of the beam diameter. Value range is 0..1 for overlap 0%..100%.
  • The value can be positive or negative depending on a particular application. The overlap parameter influences pattern's Area Calculation and Dose.

Examples


        # Create a circle pattern with overlap R of 10% and overlap T of 20%
        circle = microscope.patterning.create_circle(3e-6, 3e-6, 2e-6, 1e-6, 5e-6)
        circle.overlap_r = 0.1
        circle.overlap_t = 0.2
