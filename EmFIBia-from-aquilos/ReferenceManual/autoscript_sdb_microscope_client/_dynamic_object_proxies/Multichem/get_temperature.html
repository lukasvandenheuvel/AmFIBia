Multichem.get_temperature

Determines the temperature of the active gas, unless a specific gas is provided. The default unit is Kelvin, but you can configure a preferred unit by calling set_preferred_unit().

Syntax

get_temperature()
get_temperature(gas_name)

Parameters

gas_name str The name of the gas.

Return value

float The gas temperature.

Examples


        microscope.service.autoscript.client.set_preferred_unit(Quantity.TEMPERATURE, Unit.CELSIUS)
    
        multichem = microscope.gas.get_multichem()
        temperature = multichem.get_temperature("XeF2")
        print(temperature)
