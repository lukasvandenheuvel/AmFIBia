Multichem.gas_flow

Controls the custom gas flow for the active gas.

Data type

List[float]

Examples

Set gas flow: 

        # Obtain MultiChem object
        multichem = microscope.gas.get_multichem()
        
        # Read gas flow value
        print(multichem.gas_flow)
        
        # Set gas flow duty cycle for the active gas to 42%
        multichem.gas_flow = [42]
        
        # When a mixed gas is active (e.g. C+W),
        # set gas flow duty cycle for C to 42% and W to 38%
        multichem.gas_flow = [42, 38]

PID pressure control: 

        # Obtain MultiChem object
        multichem = microscope.gas.get_multichem()
        
        # Set gas flow pressure for the active gas to 1.5e-3 Pa
        multichem.gas_flow = [1.5e-3]
        
        # When a mixed gas is active (e.g. C+W),
        # set gas flow pressure for C to 1.5e-3 Pa and W to 2e-3 Pa
        multichem.gas_flow = [1.5e-3, 2e-3]
