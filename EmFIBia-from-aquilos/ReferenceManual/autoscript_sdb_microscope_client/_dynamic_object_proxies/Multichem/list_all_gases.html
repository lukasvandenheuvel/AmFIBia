Multichem.list_all_gases

Lists all gases in MultiChem.

Syntax

list_all_gases()
list_all_gases(include_combinations)

Parameters

include_combinations bool If set to true, the function also returns the available gas combinations.

Return value

List[str] The list of gas names.

Remarks

When the parameter include_combinations is set to true, the returned list also includes allowed combinations of available gases. The combinations use '+' as a separator. For example, a gas combination can be HMCHS+O2. Note that the order matters; HMCHS+O2 is not the same as O2+HMCHS in this scenario.

Examples

Get all available gases in MultiChem: 

        # Obtain the MultiChem object
        multichem = microscope.gas.get_multichem()
        
        # Get available gases
        gases = multichem.list_all_gases()
        
        # or
        gases = multichem.list_all_gases(False)

Get available gases including their allowed combinations: 

        # Obtain the MultiChem object
        multichem = microscope.gas.get_multichem()
        
        # Read available gases and their combinations
        gases = multichem.list_all_gases(True)
