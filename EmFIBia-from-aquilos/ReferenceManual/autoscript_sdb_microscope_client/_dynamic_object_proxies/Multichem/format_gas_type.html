Multichem.format_gas_type

A helper function that formats the gas name along with the MultiChem port identifier. You can use this function to correctly format a string that specifies both the MultiChem port and gas name, which can be assigned to the gas_type property of a pattern or used as a gas name parameter for various functions on this MultiChem object.

Syntax

format_gas_type(gas_name)

Parameters

gas_name str The name of the gas.

Return value

str The formatted string of the gas and MultiChem port.

Examples


        # The function is especially useful when constructing a gas type string for gasses from multiple MultiChem devices.
        # In the following example, we want to configure a pattern for using carbon gas from two available MultiChems on ports 1 and 4.
        
        # Get the MultiChem objects
        multichem1 = microscope.gas.get_multichem("1")
        multichem2 = microscope.gas.get_multichem("4")
        
        # Create a rectangular pattern
        rectangle_pattern = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)
        
        # Construct the gas_type string 
        gas_type = multichem1.format_gas_type("C") + multichem2.format_gas_type("C")
        
        # Note: gas_type was formatted to "M1(C);M4(C);"
        
        # Configure the pattern
        rectangle_pattern.gas_type = gas_type
