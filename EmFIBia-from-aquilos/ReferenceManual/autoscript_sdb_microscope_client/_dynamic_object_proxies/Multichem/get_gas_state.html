Multichem.get_gas_state

Provides information about the state of the active gas, unless a specific gas is provided. Possible states are listed in the MultiChemGasState enumeration.

Syntax

get_gas_state()
get_gas_state(gas)

Parameters

gas str The name of the gas. If not specified, the property returns the state of the active gas.

Return value

str The state of the gas, for example "Heating".

Examples


        # Obtain MultiChem object
        multichem = microscope.gas.get_multichem()
        
        # Read gas state for the active gas
        print(multichem.get_gas_state())
        
        # Read gas state for a specific gas, e.g. "W"
        print(multichem.get_gas_state("W"))
        
        # Open gas flow if ready 
        if multichem.get_gas_state() == MultiChemGasState.READY:
            multichem.open()
