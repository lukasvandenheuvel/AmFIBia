Multichem.turn_heater_off

Turns the MultiChem heater off for the active gas, unless a specific gas is provided.

Syntax

turn_heater_off()
turn_heater_off(gas_name)

Parameters

gas_name str The name of the gas.

Examples


        # Obtain MultiChem object
        multichem = microscope.gas.get_multichem()
        
        # Turn off heater for the active gas
        multichem.turn_heater_off()
        
        # Turn off heater for a specific gas (e.g. XeF2).
        # The active gas remains the same.
        multichem.turn_heater_off("XeF2")
        
        # Turn off heater for a specific mixed gas (e.g. C+W).
        # The active gas remains the same.
        multichem.turn_heater_off("C+W")
