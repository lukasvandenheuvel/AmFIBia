Multichem

Represents the MultiChem gas injection device.

Properties

active_gas Retrieves or assigns the current gas on the MultiChem port.
available_needle_positions [Read only] Returns a list of available needle positions. Each position is described by its label, assigned during microscope alignment or from the main user interface application. Default positions are listed in the MultiChemInsertPosition enumeration.
gas_flow Controls the custom gas flow for the active gas.
injector_state [Read only] Returns the state of MultiChem injector. The MultiChemInjectorState enumeration lists all possible states.
needle_position [Read only] Returns the name of the current position of the MultiChem needle. The available positions were designated during microscope alignment, numeric coordinates are not accessible via the API.
port_name [Read only] Returns the name of the port to which this MultiChem device is connected.
state [Read only] Returns the state of the MultiChem needle. The RetractableDeviceState enumeration lists all possible states.

Methods

close() Closes the gas flow in MultiChem.
format_gas_type() A helper function that formats the gas name along with the MultiChem port identifier. You can use this function to correctly format a string that specifies both the MultiChem port and gas name, which can be assigned to the gas_type property of a pattern or used as a gas name parameter for various functions on this MultiChem object.
get_gas_state() Provides information about the state of the active gas, unless a specific gas is provided. Possible states are listed in the MultiChemGasState enumeration.
get_temperature() Determines the temperature of the active gas, unless a specific gas is provided. The default unit is Kelvin, but you can configure a preferred unit by calling set_preferred_unit().
insert() Inserts the MultiChem needle into one of the predefined positions.
list_all_gases() Lists all gases in MultiChem.
open() Opens the flow of the active gas, unless a specific gas is provided. This action also sets the gas as active.
purge() Initiates the purge procedure for the active gas, unless a specific gas is provided. This action also sets the gas as active.
retract() Retracts the MultiChem needle if inserted.
turn_heater_off() Turns the MultiChem heater off for the active gas, unless a specific gas is provided.
turn_heater_on() Turns the MultiChem heater on for the active gas, unless a specific gas is provided.
