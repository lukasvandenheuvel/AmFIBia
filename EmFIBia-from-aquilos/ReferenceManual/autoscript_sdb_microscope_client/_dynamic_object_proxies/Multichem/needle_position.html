Multichem.needle_position

Returns the name of the current position of the MultiChem needle. The available positions were designated during microscope alignment, numeric coordinates are not accessible via the API.

Data type

[Read only] str

Examples


        # Obtain MultiChem object
        multichem = microscope.gas.get_multichem()
        
        # Check if the needle is in the correct position
        if multichem.needle_position != MultiChemInsertPosition.COMPUSTAGE90_DEFAULT:        
            # If it's not, adjust the needle's position accordingly
            multichem.insert(MultiChemInsertPosition.COMPUSTAGE90_DEFAULT)
