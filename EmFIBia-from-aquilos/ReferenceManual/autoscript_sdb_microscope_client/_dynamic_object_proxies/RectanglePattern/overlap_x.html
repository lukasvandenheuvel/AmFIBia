RectanglePattern.overlap_x

Sets the beam diameter overlap in X direction.

Data type

float

Remarks

  • The overlap value is specified as percentage of the beam diameter. Value range is 0..1 for overlap 0%..100%.
  • The value can be positive or negative depending on a particular application. The overlap parameter influences pattern's Area Calculation and Dose.

Examples


        # Create a rectangular pattern with overlap X of 40% and overlap Y of 60%
        rect = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)
        rect.overlap_x = 0.4
        rect.overlap_y = 0.6
