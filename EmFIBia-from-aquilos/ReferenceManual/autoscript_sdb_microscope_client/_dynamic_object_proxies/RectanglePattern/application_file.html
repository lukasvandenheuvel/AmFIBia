RectanglePattern.application_file

The name of the application file.

Data type

str

Remarks

You can set the default application file that will be used for every newly created pattern using the patterning.set_default_application_file() function.

Examples


        # Create a rectangular pattern
        pattern = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)
        
        # Configure the pattern with the "Enh etch" application file
        pattern.application_file = "Enh etch"
