RectanglePattern.gas_type

The name of the gas required to pattern this shape, or an empty string if no gas should be used.

Data type

str

Remarks

To ensure proper formatting of the gas type, you can utilize the Multichem.format_gas_type() function.

Examples

Single MultiChem situation: 

        rectangle_pattern = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)
        
        # Assign a single gas with a MultiChem
        rectangle_pattern.gas_type = "XeF2"
        
        # Set gas duty cycle to 10%
        rectangle_pattern.gas_flow = [10]
        
        # Assign a mixed gas with a MultiChem
        rectangle_pattern.gas_type = "C+Dx"
        
        # Set gas duty cycle to 10, 90%
        rectangle_pattern.gas_flow = [10, 90]
Multiple MultiChems situation: 

        # Create a rectangular pattern 
        rectangle_pattern = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)
        
        # Assign it a single gas with a single MultiChem 
        multichem = microscope.gas.get_multichem("1")
        rectangle_pattern.gas_type = multichem.format_gas_type("C")
        
        # Set gas duty cycle to 15% 
        rectangle_pattern.gas_flow = [15]
        
        # Assign it a mixed gas with a single MultiChem 
        multichem = microscope.gas.get_multichem("4")
        rectangle_pattern.gas_type = multichem.format_gas_type("XeF2+Dx")
        
        # Set gas duty cycle to 15.5%, 84.5% 
        rectangle_pattern.gas_flow = [15.5, 84.5]
        
        # Assign it a single gas with two MultiChems 
        multichem1 = microscope.gas.get_multichem("1")
        multichem2 = microscope.gas.get_multichem("4")
        rectangle_pattern.gas_type = multichem1.format_gas_type("C") + multichem2.format_gas_type("C")
        
        # Set gas duty cycle to 10%, 90% 
        rectangle_pattern.gas_flow = [10, 90]
Situation when PID gas flow control is enabled: 

        rectangle_pattern = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)
        
        # Assign a single gas with MultiChem
        multichem = microscope.gas.get_multichem()
        rectangle_pattern.gas_type = multichem.format_gas_type("W")
        
        # Set gas pressure to 2.5e-3 Pa
        rectangle_pattern.gas_flow = [2.5e-3]
    

        rectangle_pattern = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)
        
        # Assign multiple gases with MultiChem
        multichem = microscope.gas.get_multichem()
        rectangle_pattern.gas_type = multichem.format_gas_type("C+W")
        
        # Set gas pressure to 2.8e-3 Pa
        rectangle_pattern.gas_flow = [2.8e-3]
