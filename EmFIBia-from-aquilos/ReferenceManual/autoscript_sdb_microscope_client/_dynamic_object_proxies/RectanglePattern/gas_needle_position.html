RectanglePattern.gas_needle_position

The position of the MultiChem needle when patterning begins.

Data type

str

Remarks

MultiChem labeled positions are case-sensitive. Double-check when assigning the gas needle position using a string literal to ensure correctness.

Examples


    # Create a pattern
    pattern = microscope.patterning.create_rectangle(0, 0, 1e-6, 1e-6, 1e-6)
    
    # Get the MultiChem object
    multichem = microscope.gas.get_multichem()
    
    # Assign a MultiChem needle position to the pattern
    pattern.gas_needle_position = multichem.available_needle_positions[1]
    
    # Or assign it by a string
    if "Offset X 10um" in multichem.available_needle_positions:
    	pattern.gas_needle_position = "Offset X 10um"
