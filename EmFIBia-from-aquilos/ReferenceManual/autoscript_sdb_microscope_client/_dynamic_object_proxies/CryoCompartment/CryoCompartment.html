CryoCompartment

Represents the cryo compartment device.

Properties

dewar_level [Read only] Retrieves the value of Dewar level within the range of [0, 1] corresponding to 0% - 100%.
state [Read only] Retrieves the current temperature state of the compartment, returning a value from the CryoCompartmentTemperature enumeration.

Methods

go_to_cryo_temperature() Runs a cooling procedure that lowers the compartment temperature to cryo temperature.
go_to_room_temperature() Runs a heating procedure that raises the compartment temperature to room temperature.
refill_dewar() Refills the Dewar for the compartment.
run_cryo_cycle() Runs a cryo cycle procedure. At the end of the procedure, the compartment is set to the specified temperature.
