CryoCompartment.refill_dewar

Refills the Dewar for the compartment.

Syntax

refill_dewar()

Remarks

The refill_dewar() procedure is enabled on the Autoloader device when the Dewar level is below 70% and the compartment is at cryo temperature.

Examples


    # Refill Dewar if it is under 40% 
    if compartment.dewar_level < 0.40:
        compartment.refill_dewar()
