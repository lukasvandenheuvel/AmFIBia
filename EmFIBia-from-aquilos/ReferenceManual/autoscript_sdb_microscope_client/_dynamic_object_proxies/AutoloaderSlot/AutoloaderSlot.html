AutoloaderSlot

Represents an object of the Autoloader slot and provides access to its properties.

Properties

id [Read only] Returns the slot ID. The ID refers to the numerical identifier used for loading or unloading samples.
sample_description Retrieves or updates the description of the sample attached to the slot.
state [Read only] Retrieves the current occupation state of the Autoloader slot. The possible values for the occupation state are "Occupied", "Empty", or "Unknown".
