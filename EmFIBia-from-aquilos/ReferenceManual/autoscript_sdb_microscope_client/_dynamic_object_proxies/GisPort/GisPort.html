GisPort

Represents a single Gas Injection System (GIS) port.

Methods

close() Closes the GIS port.
get_temperature() Gets the GIS port gas temperature. The default unit is Kelvin, but you can configure a preferred unit by calling set_preferred_unit().
insert() Inserts the GIS needle if it is retracted.
open() Opens the GIS port.
retract() Retracts the GIS needle if it is inserted.
turn_heater_off() Turns the GIS heater off.
turn_heater_on() Turns the GIS heater on.
