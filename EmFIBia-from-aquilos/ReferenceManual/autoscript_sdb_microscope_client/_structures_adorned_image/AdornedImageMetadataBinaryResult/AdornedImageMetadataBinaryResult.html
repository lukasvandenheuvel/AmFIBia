AdornedImageMetadataBinaryResult

Section of image metadata that describes the properties of the resulting image.

Items

acquisition_unit [Optional] str The basic unit of acquisition as one of CameraImage/Pixel/Spectrum/Invalid.
bits_per_pixel [Optional] int The number of bits per pixel. Note encoding for precision. Unsigned 8 would mean grayscale 8 bits per pixel, RGB 24 would mean 8 bits per color.
black_level [Optional] float Luminosity value in the image to display as the "black" (darkest) level.
composition_type [Optional] str The method used to compose the acquisition units.
detector [Optional] str The name of the detector that generated the image.
detector_index [Optional] int The index of the detector in the list of detectors.
digital_brightness [Optional] float Brightness which is applied to the image after acquisition and prior to saving.
digital_contrast [Optional] float Contrast which is applied to the image after acquisition and prior to saving.
encoding [Optional] str Encoding type for raw image.
filter_frame_count [Optional] int Number of source frames combined using the averaging or integration filter.
filter_type [Optional] str The type of filter used, for instance, integration or averaging filter.
gamma [Optional] float Image adjustment gamma setting. Default of 1.0 = no adjustment.
intensity_offset [Optional] float The value added to the scaled input value to determine the actual intensity. Default is IntensityOffset = 0.0. AdjustedIntensity = (IntensityScale * InputValue) + IntensityOffset.
intensity_scale [Optional] float Multiplication factor to convert intensity values to other physical measurement in the event they do not represent intensity. Default intensity scale = 1.0.
pixel_value_maximum [Optional] float Computed maximum numeric value of pixel values of the unprocessed image. For grayscale images corresponds to the white-most value.
pixel_value_minimum [Optional] float Computed minimum numeric value of pixel values of the unprocessed image. For grayscale images corresponds to the black-most value.
pixel_value_mean [Optional] float Computed average (mean) numeric value of pixel values of the unprocessed (grayscale) image.
pixel_value_standard_deviation [Optional] float Computed standard deviation numeric value of pixel values of the unprocessed image.
recursive_filter_coefficient [Optional] float User-selected value that describes the filtering.
reference_transformation [Optional] str The reference transformation matrix translates pixel coordinates to the coordinate system at the optical axis/focus plane. The unit can be m or rad.
sharpness [Optional] float Sharpness as computed from the image pixel data. Future consideration is needed for algorithm definition.
sharpness_algorithm [Optional] str The algorithm used to calculate the sharpness value.
sigma [Optional] float Sigma adjustment.
white_level [Optional] float Luminosity value in the image to display as the "white" (brightest) level.
acquisition_area Get: [Optional] Rectangle
Set: [Optional] Rectangle | list | tuple		 The relative area from the full potential area used for acquisition to arrive at ImageSize and Offset. Range 0.0 to 1.0.
image_size Get: [Optional] Point
Set: [Optional] Point | list | tuple		 The image size in pixels.
offset Get: [Optional] Point
Set: [Optional] Point | list | tuple		 Relative distance between the top-left (0, 0) pixel and the geometric center of the scanned area, measured in meters.
pixel_size Get: [Optional] Point
Set: [Optional] Point | list | tuple		 Physical size of the image pixel.
