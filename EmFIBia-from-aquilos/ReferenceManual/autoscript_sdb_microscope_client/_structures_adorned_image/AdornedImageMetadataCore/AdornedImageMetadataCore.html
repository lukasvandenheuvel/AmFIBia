AdornedImageMetadataCore

Section of image metadata that describes software environment on the microscope computer.

Items

application_computer_name [Optional] str Represents the computer that the application software is running on.
application_software [Optional] str The application the user interacts with to setup the acquisition; application that saves the image.
application_software_version [Optional] str 4-part version number: #.#.#.#.
comment [Optional] str User provided description or comment.
file_date_time [Optional] str Date-time when the result file is created.
file_name [Optional] str Filename and extension of the pixel content (binary data file) described by this metadata.
guid [Optional] str Used internally within Thermo Fisher Scientific systems as the unique ID of the image instance.
md5_checksum [Optional] bytes MD5 hash of binary result data. Used to determine duplicate results (when importing), or modification (checksum).
parent_guid [Optional] str Thermo Fisher Scientific system ID of the image from which this data was derived.
private_checksum [Optional] bytes Vendor-specific checksum used for proprietary purposes such as validating the image was from a particular instrument manufacturer.
user_id [Optional] str The user ID or user name of the user running the process that acquired the image.
