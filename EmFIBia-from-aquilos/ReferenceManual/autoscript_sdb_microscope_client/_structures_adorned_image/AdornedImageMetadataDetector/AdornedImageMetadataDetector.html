AdornedImageMetadataDetector

Section of image metadata related to the detector used for image acquisition.

Items

detector_name [Optional] str The abbreviated name of the detector.
detector_type [Optional] str The type of the detector.
enabled [Optional] bool True if the detector was enabled during image acquisition.
inserted [Optional] bool True if the detector was inserted into the specimen chamber during image acquisition.
signal [Optional] str Detector mode, for example backscatter electrons (BSE) or secondary electrons (SE).
gain [Optional] float Detector gain in decibels. For ESEM this is enhanced gain. To be distinguished from digital contrast.
contrast_normalized [Optional] float Contrast voltage converted to decibels then normalized to a 0-100 user-facing value.
offset [Optional] float Detector offset in volts.
brightness_normalized [Optional] float Value applied to detector inducing brightness then converted to a 0-100 user-facing value.
