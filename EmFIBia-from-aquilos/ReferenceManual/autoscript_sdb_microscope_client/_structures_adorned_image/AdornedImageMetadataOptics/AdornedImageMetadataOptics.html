AdornedImageMetadataOptics

Section of image metadata explaining the optical parameters of the electron or ion beam column.

Items

acceleration_voltage [Optional] float The accelerating voltage for the charged particle.
apertures List[AdornedImageMetadataOpticsAperture] The list of available apertures.
beam_convergence [Optional] float Convergence semi-angle of the beam.
beam_current [Optional] float The actual current induced into the sample.
beam_current_selected [Optional] float The user-selected beam current which may vary from actual beam current for purposes of querying and report generation.
beam_diameter [Optional] float Diameter of the beam at the sample, in meters. Logically similar to SpotSize, but applies to a different mode, so not comparable.
camera_length [Optional] float "Length" of a virtual camera system (distance between the film plate/CCD and the sample). Only for diffraction.
deceleration_voltage [Optional] float Deceleration voltage [V]. Landing Energy will be calculated based on AccelerationVoltage minus DecelerationVoltage. ImmersionRatio will also be calculated as AccelerationVoltage divided by LandingEnergy.
defocus [Optional] float The distance from the focus plane to a plane the user chooses, in meters.
emission_current [Optional] float The source emission current.
eucentric_working_distance [Optional] float The working distance of the primary beam to the beam intersection.
extractor_voltage [Optional] float Voltage applied to the extractor electrode to extract electrons from the tip, in volts. FEG only.
fib_l0_voltage [Optional] float Voltage applied to L0 which contributes to the Focus. Only applies to some FIB systems.
fib_l1_voltage [Optional] float Voltage applied to L1 which contributes to Beam Current and Focus.
fib_l2_voltage [Optional] float Voltage applied to L2 which contributes to the Focus.
focus [Optional] float Distance from the focus plane to the eucentric focus plane, in meters. Controlled by the objective lens.
full_scan_field_of_view [Optional] AdornedImageMetadataOpticsScanFieldSize The size of the full scan, in meters.
landing_energy [Optional] float The beam landing energy is be calculated based on AccelerationVoltage minus DecelerationVoltage.
last_measured_screen_current [Optional] float The last measured flu-screen current when the screen was inserted.
sample_pre_tilt_angle [Optional] float The inherent sample tilt. The combined sample tilt and stage tilt will be sent to optics for tilt correction.
scan_field_of_view [Optional] AdornedImageMetadataOpticsScanFieldSize The size of the image as determined by the physical size of the scan system. The image field of view is always smaller or equal to the full scan field of view.
screen_current [Optional] float The current as measured on the flu-screen, in amperes.
spherical_aberration [Optional] float The spherical aberration of the objective lens or the spherical aberration set by the aberration corrector, in meters.
spot_size [Optional] float The spot diameter.
stem_focus [Optional] float Distance from the focus plane to the eucentric focus plane (STEM), in meters. Controlled by changing intensity or objective.
wehnelt_bias [Optional] float The bias voltage of the Wehnelt, in volts. Thermionic systems only.
working_distance [Optional] float The distance from the pole tip to the focal point of the beam.
cross_over_on [Optional] bool True if cross over mode is on. Relates to ScanMode in SEM.
diffraction_focus [Optional] float The focus value used in diffraction.
eftem_on [Optional] bool True if magnifications are adapted to the energy filter.
gun_filament_settings [Optional] float The setting of the filament heating current.
gun_lens_setting [Optional] float The user selected value that affects the voltage of the gun lens.
high_magnification_mode [Optional] str High magnification sub-modes.
illumination_intensity_normalized [Optional] float Illumination intensity normalized to a range of 0-100, where 100 represents the maximum possible intensity for the illumination source and 0 represents none.
illumination_mode [Optional] str Illumination/Condenser mode.
illumination_on [Optional] bool True if the optical camera illumination source is active during image capture.
illumination_type [Optional] str The name of the type of illumination.
nominal_magnification [Optional] float Magnification to the chosen camera plane.
objective_lens_mode [Optional] str Mode of the objective lens. Aka LensMode, ObjectiveMode. STEM Only.
operation_mode [Optional] str Primary operating mode of the column.
optical_mode [Optional] str The name of the optical preset.
probe_mode [Optional] str Mode of the minicondenser probe.
projector_mode [Optional] str Mode of the projection system.
sample_tilt_correction_is_on [Optional] bool True if tilt-correction is turned on.
spot_index [Optional] int SpotIndex is a number that indicates the order of the beam current. Spot Index 1 is the highest beam current. The beam current reduces as inverse of the square, where higher SpotIndex numbers indicate the progressive reduction in the beam current.
tem_operating_sub_mode [Optional] str TEM specific operating mode of the column.

c2_lens_intensity [Optional] float C2 lens setting. TEM only. To get as a percentage, divide the value by its maximum range value, in most cases 0.902.
c3_lens_intensity [Optional] float C3 lens setting. TEM only. To get as a percentage, divide the value by its maximum range value, in most cases 0.902.
condenser_stigmator_raw Get: [Optional] Point
Set: [Optional] Point | list | tuple		 Stigmator of the beam shape. To be distinguished from the GunStigmator.
diffraction_lens_intensity [Optional] float Diffraction lens setting. TEM only. To get as a percentage, divide the value by its maximum range value, in most cases 0.902.
diffraction_stigmator_raw Get: [Optional] Point
Set: [Optional] Point | list | tuple		 The uncalibrated value of the diffraction lens stigmator.
