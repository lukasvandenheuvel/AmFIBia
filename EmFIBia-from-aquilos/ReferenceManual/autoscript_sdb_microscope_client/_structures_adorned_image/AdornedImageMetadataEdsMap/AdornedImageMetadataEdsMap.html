AdornedImageMetadataEdsMap

Section of image metadata describing the result of the EDS elemental or phase mapping experiment.

Items

engine_type [Optional] str The type of EDS elemental or phase mapping engine. A value from EdsMappingEngineType enumeration.
segmentation_method [Optional] str The type of image segmentation algorithm for the EDS elemental mapping engine.
segmentation_resolution [Optional] str The resolution setting for image segmentation in the EDS phase mapping engine.
filter_type [Optional] str The filter setting for reducing the noise in the input data within the EDS phase mapping engine.
filter_mask [Optional] str The image filter strength for image segmentation within the EDS phase mapping engine
element_names [Optional] List[str] Comma-separated list of chemical element names captured in the image, which is an EDS elemental map.
element_colors [Optional] List[int] Comma-separated list of colors in hexadecimal format of the elements captured in the image, which is an EDS elemental map.
phase_indices [Optional] List[int] Comma-separated phase indices captured in the image, which is an EDS phase map.
phase_colors [Optional] List[int] Comma-separated list of colors in hexadecimal format of the phases captured in the image, which is an EDS phase map.
