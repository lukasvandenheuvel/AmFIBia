AdornedImage

Contains microscope image data along with metadata describing the state of the microscope under which the image was acquired.

Items

width [Read only] int The width of the image.
height [Read only] int The height of the image.
bit_depth [Read only] int The bit depth of the image data.
metadata [Optional] AdornedImageMetadata Metadata containing information about system state at the time this image was acquired.
data [Read only] Numpy.ndarray Image data arranged either in 2D (for grayscale images) or 3D (for RGB images) array.
encoding [Read only] int Encoding of the image. Accepts and returns values from the ImageDataEncoding enumeration.
checksum [Read only] int Checksum of the raw-data.
thumbnail [Read only] AdornedImage Thumbnail of the original image.

Methods

load() Loads an image from a TIFF file.
save() Saves the image at the specified path as a TIFF file.

Remarks

The new AdornedImage can be constructed either empty or from the given data if passed. The data has to be in a format of a numpy.ndarray.

Examples


        import cv2
        from matplotlib import pyplot as plot
        
        # Take one image with scan settings currently set in the active view
        image_original = microscope.imaging.grab_frame()
        
        # Threshold the image using OpenCV
        thresh, data_processed = cv2.threshold(image_original.data, 127, 255, cv2.THRESH_BINARY)
        
        # Construct an adorned image object from the processed data
        image_processed = AdornedImage(data_processed)
        
        # Recommended: Construct an adorned image object from the processed data reusing the original metadata
        image_processed = AdornedImage(data_processed, image_original.metadata)
        
        # Display the processed image in a window
        plot.imshow(image_processed.data, cmap="gray")
        plot.show()
