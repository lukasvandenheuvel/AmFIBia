AdornedImage.thumbnail

Thumbnail of the original image.

Data type

[Read only] AdornedImage

Remarks

The thumbnail is created by resizing the original image to a maximum of 512 pixels in each dimension, while maintaining the aspect ratio.
The exact formula is 

        thumbnail_width, thumbnail_height = (round(self.width * 512 / self.height), 512) \
            if self.width < self.height else (512, round(self.height * 512 / self.width))
