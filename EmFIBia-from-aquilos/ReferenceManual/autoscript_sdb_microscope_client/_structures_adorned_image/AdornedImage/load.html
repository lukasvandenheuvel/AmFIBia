AdornedImage.load

Loads an image from a TIFF file.

Syntax

load(path)

Parameters

path The path to the TIFF file from which to load the image.

Return value

AdornedImage Loaded image.

Remarks

  • If the source image has metadata, the function reads a small subset of the metadata and initializes members under AdornedImage.metadata. In the current AutoScript version, only the beam type, field of view, and pixel size metadata values are initialized.
  • TIFF is a preferred image format because it allows metadata to be stored with the image. However, the function can also read BMP , PNG and JPG image formats. For those other formats, the member AdornedImage.metadata is left uninitialized.
