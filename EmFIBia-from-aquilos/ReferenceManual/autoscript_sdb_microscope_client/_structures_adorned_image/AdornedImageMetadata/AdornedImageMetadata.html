AdornedImageMetadata

Image metadata that captures the state of the microscope at the time of image acquisition.

Items

acquisition AdornedImageMetadataAcquisition Section of image metadata related to image acquisition.
binary_result AdornedImageMetadataBinaryResult Section of image metadata that describes the properties of the resulting image.
core AdornedImageMetadataCore Section of image metadata that describes the software environment on the microscope computer.
detectors List[AdornedImageMetadataDetector] Section of image metadata related to the detectors used for image acquisition.
energy_filter_settings AdornedImageMetadataEnergyFilterSettings Section of image metadata describing properties of the electron or ion beam.
gas_injection_systems List[AdornedImageMetadataGasInjectionSystem] Section of image metadata describing the state of the gas injection system.
instrument AdornedImageMetadataInstrument Section of image metadata describing the properties of the microscope system and its software environment.
optics AdornedImageMetadataOptics Section of image metadata explaining the optical parameters of the electron or ion beam column.
sample AdornedImageMetadataSample Section of image metadata describing the properties of the sample.
scan_settings AdornedImageMetadataScanSettings Section of image metadata describing scan settings.
stage_settings AdornedImageMetadataStageSettings Section of image metadata describing stage settings.
vacuum_properties AdornedImageMetadataVacuumProperties Section of image metadata describing vacuum properties.
metadata_as_ini str Metadata encoded in a string with the key-value structure. May be empty if the image source supplied metadata only in xml format.
metadata_as_xml str Metadata encoded in a xml-formatted string. May be empty if the image source supplied metadata only in INI (key-value) format.
