AdornedImageMetadataScanSettings

Section of image metadata describing scan settings.

Items

scan_rotation [Optional] float The rotation angle of the scan.
line_integration_count [Optional] int The number of lines that are averaged before proceeding to the next line. The default value is 1.
line_interlacing [Optional] int The number of partial frames to get one total frame. The default value is 1.
mains_lock_on [Optional] bool True if the image line or frame scan frequency is synchronized with the power supply frequency.
scan_size [Optional] str Total size of the frame width and height, in pixels.
acquisition_time [Optional] float The total time taken to acquire the image/result. Always >= the frame time, but may be larger due to post processing or multi-frame compositing.
dwell_time [Optional] float Time to scan one pixel, in seconds.
frame_time [Optional] float Time to scan one frame, including flyback times, in seconds.
line_time [Optional] float Time to scan one line including flyback time, in seconds.
scan_area Get: [Optional] Rectangle
Set: [Optional] Rectangle | list | tuple		 The rectangular area of the scan. The actual captured area, which must be <= the Scan Size. If the ScanArea is not provided then it is assumed to be the Scan Size with 0,0 for X, Y.
