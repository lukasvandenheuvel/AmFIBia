StreamPatternDefinition.load

Loads the stream pattern definition from a stream (*.str) file.

Syntax

load(path)

Parameters

path The path to the stream file from which to load the stream pattern definition.

Return value

StreamPatternDefinition The loaded stream pattern definition.

Remarks

The file must be in the proprietary "stream file" format from Thermo Fisher Scientific. The file specifies mill point positions, beam dwell time, and blanking flags in a simple text file. This format is also supported by the primary user interface application, Microscope Control.
