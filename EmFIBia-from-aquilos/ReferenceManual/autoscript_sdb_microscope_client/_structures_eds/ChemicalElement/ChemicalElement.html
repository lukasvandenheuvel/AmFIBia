ChemicalElement

Represents one chemical element.

Items

symbol [Optional] str The symbol of the chemical element as it appears in the periodic table.
name [Optional] str The name of the chemical element.
atomic_number [Optional] int The number of protons in the nucleus of each atom of the element.
line_family [Optional] str The spectral line as one of the strings "K", "L" or "M". The line indicates the energy level of X-rays from which the presence of this chemical element is determined. The line identifier is only considered by EDS mapping functions.
K [Optional] ChemicalElement Returns a copy of this chemical element with the selected spectral line "K".
L [Optional] ChemicalElement Returns a copy of this chemical element with the selected spectral line "L".
M [Optional] ChemicalElement Returns a copy of this chemical element with the selected spectral line "M".

Remarks

AutoScript defines an auxiliary class PeriodicTable that holds a collection of all elements from the periodic table as instances of the ChemicalElement class. The class is especially useful when you want to pass chemical elements as parameters to the spectrum processing functions analysis.get_peak_id() or analysis.eds.get_composition().

Examples


        # Create a chemical element object that represents nitrogen
        nitrogen = ChemicalElement("N")
        
        # ...or simply 
        nitrogen = PeriodicTable.NITROGEN
        
        # Create a chemical element object representing titanium with "L" spectral line specified
        titanium_l = PeriodicTable.TITANIUM.L
