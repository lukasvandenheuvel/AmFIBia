EdsPhaseAnalysisSettings

Contains settings for the EDS phase analysis, which is conducted by the function run_phase_analysis().

Items

component_count [Optional] int The number of components (and therefore phases) to search for in the input EDS stream during the phase analysis. If not provided or set to 0, the analysis will determine the number of components automatically.
segmentation_resolution [Optional] str The resolution of the image segmentation layers inside the phase mapping engine. The segmentation algorithm processes component maps and generates binary phase maps. The resolution does not affect the size of the resulting map, which is always either 768x512 or 512x442 depending on the scanning resolution aspect ratio.
filter_type [Optional] str The type of image filter used internally by the phase mapping engine to process component maps into binary phase maps.
filter_mask [Optional] str The strength of the image filter used internally by the phase mapping engine to process component maps into binary phase maps.
