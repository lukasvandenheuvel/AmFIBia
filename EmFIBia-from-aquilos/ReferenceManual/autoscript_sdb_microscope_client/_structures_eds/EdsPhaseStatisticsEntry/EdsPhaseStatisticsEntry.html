EdsPhaseStatisticsEntry

Contains statistical data for a single phase identified in an EDS stream. The function run_phase_analysis() returns a list of these structures for each phase it identifies in the input stream.

Items

phase_index [Optional] int An integer representing the index of the phase, starting from 1.
phase_name [Optional] str A string identifier for the phase.
area_in_meters [Optional] float The area the phase occupies in the field of view, measured in square meters.
area_in_pixels [Optional] int The area the phase occupies in the field of view, measured in pixels.
area_in_percent [Optional] float The area the phase occupies in the field of view, expressed as a percentage of the entire site area.
