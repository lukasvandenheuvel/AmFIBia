PeriodicTable

An auxiliary class representing the periodic table of chemical elements. The main purpose of the class is to provide preconfigured elements as instances of ChemicalElement class that can be easily passed as arguments to EDS spectrum processing functions.

Examples


    # Get spectrum collected by previous scanning activity 
    spectrum = microscope.analysis.eds.get_spectrum()
    
    # Identify elements, removing carbon and oxygen from the result
    peaks = microscope.analysis.eds.get_peak_id(spectrum, [PeriodicTable.CARBON, PeriodicTable.OXYGEN])
