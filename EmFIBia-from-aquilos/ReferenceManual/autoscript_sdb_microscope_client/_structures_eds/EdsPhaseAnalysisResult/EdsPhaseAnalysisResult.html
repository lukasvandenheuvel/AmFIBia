EdsPhaseAnalysisResult

Holds the outcomes of phase analysis conducted on the EDS stream, retrieved from the function run_phase_analysis().

Items

component_count [Optional] int The count of chemical components (and therefore phases) identified within the EDS stream. If a specific number of components was provided as an input parameter for the analysis, then this count corresponds to the specified input. However, if no number was specified, the analysis automatically detects the components.
statistics [Optional] List[EdsPhaseStatisticsEntry] Encapsulates statistical data for individual phases, presented as a list of EdsPhaseStatisticsEntry structures. Each entry furnishes details about a single identified phase, primarily concerning the area it occupies within the field of view.

Methods

get_all_phases() Serves as a utility to generate a list of phase indices ranging from 1 to ComponentCount. This list is commonly utilized as a parameter for the function get_phase_map() to retrieve the composite map image comprising all phases.
