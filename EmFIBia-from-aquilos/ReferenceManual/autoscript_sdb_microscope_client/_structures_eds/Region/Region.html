Region

Specifies the area for image or spectrum acquisition.

Items

type str Type of the region. You can use values from the RegionType enumeration.
point Get: [Optional] Point
Set: [Optional] Point | list | tuple		 The X and Y coordinate of the point, in pixels by default. Must be specified for POINT regions.
rectangle Get: [Optional] Rectangle
Set: [Optional] Rectangle | list | tuple		 The position and size of the rectangle, in pixels by default. Must be specified for RECTANGLE regions.
vertices Get: [Optional] List[Point]
Set: [Optional] List[Point | list | tuple]		 The list of polygon vertices as points, where each point specifies X and Y coordinates of the vertex, in pixels by default. The minimum number of vertices is 3. Must be specified for POLYGON regions.
coordinate_system [Optional] str Coordinate system in which the region coordinates are specified. You can use values from the RegionCoordinateSystem enumeration.

Remarks

In EDS spectrum acquisition, namely the analysis.eds.acquire_spectrum() function, the concept of regions is used to delineate the area of the sample where scanning is to occur, generating the EDS signal. The Region structure is first assigned a type and based on the type additional parameters are entered to define the position and size of the region.

Regions unify different coordinate systems of the underlying imaging and patterning objects (such as reduced area, spot, polygon pattern) into one - pixels. All coordinates specified in pixels are relative to the currently set scanning resolution of the electron beam (beams.electron_beam.scanning.resolution.value property). Coordinate [0, 0] is the upper left corner of the field of view, [resolution X, resolution Y] is the lower right corner.

Starting from AutoScript version 4.8.0 regions can be defined in other than pixel coordinate systems, see RegionCoordinateSystem enumeration.
The default coordinate system is Pixels if none is explicitly specified in the region constructor.

Examples


        # Create a region for scanning across the entire field of view
        region = Region(type=RegionType.FULL_FRAME)
       
        # Create a region for scanning over a rectangular part of the field of view, given in pixels
        region = Region(type=RegionType.RECTANGLE, rectangle=Rectangle(100, 100, 400, 400))
        
        # Create a region for scanning over a rectangular part of the field of view, given in reduced area coordinate system
        region = Region(type=RegionType.RECTANGLE, rectangle=[0.1, 0.2, 0.4, 0.4],
                        coordinate_system=RegionCoordinateSystem.REDUCED_AREA)

        # Create a region for scanning over a single point
        region = Region(type=RegionType.POINT, point=[200, 300])
        
        # Create a region for scanning over a polygon defined by coordinates of its vertices  
        region = Region(type=RegionType.POLYGON, vertices=[Point(10, 10), Point(200, 200), Point(20, 400)])
