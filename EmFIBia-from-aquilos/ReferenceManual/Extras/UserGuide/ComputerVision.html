User guide: Computer vision

In this chapter, you will learn how to use the computer vision features of the AutoScript toolkit Vision and User interface packages.

Before proceeding with the code snippets below, initialize AutoScript with the following code. Set the server address according to your environment. 


    from autoscript_sdb_microscope_client import SdbMicroscopeClient
    from autoscript_sdb_microscope_client.enumerations import *
    from autoscript_sdb_microscope_client.structures import *
    
    microscope = SdbMicroscopeClient()
    microscope.connect("localhost")

Import Vision and User interface packages with handy aliases:


    import autoscript_toolkit.vision as vision_toolkit
    import autoscript_toolkit.user_interface as ui_toolkit

Prepare a function to plot images:


    from matplotlib import pyplot as plot
    
    def quick_plot(image):
        plot.imshow(image.data, cmap='gray')
        plot.show()

Cut out image region using relative coordinates, plot the original image and the template image:


    settings = GrabFrameSettings(resolution="1536x1024", dwell_time=1e-6)
    original_image = microscope.imaging.grab_frame(settings)
    
    template_image = vision_toolkit.cut_image(original_image, relative_center=[0.40, 0.40], relative_size=[0.20, 0.20])
    
    quick_plot(original_image)
    quick_plot(template_image)

Cut out image region using pixel coordinates, plot the original image and the template images:


    settings = GrabFrameSettings(resolution="1536x1024", dwell_time=1e-6)
    original_image = microscope.imaging.grab_frame(settings)
    
    template_image_1 = vision_toolkit.cut_image(original_image, center=[140, 96], size=[120, 120])
    template_image_2 = vision_toolkit.cut_image(original_image, top_left=[80, 36], bottom_right=[200, 156])
    
    quick_plot(original_image)
    quick_plot(template_image_1)
    quick_plot(template_image_2)

Locate a template in the image using default template matcher and inspect the result:


    # Take a full-frame image and cut out a template image from it
    settings = GrabFrameSettings(resolution="1536x1024", dwell_time=1e-6)
    image = microscope.imaging.grab_frame(settings)
    template = vision_toolkit.cut_image(image, relative_center=[0.40, 0.40], relative_size=[0.20, 0.20])
    
    # Locate the template in the original image
    location = vision_toolkit.locate_feature(image, template)
    ui_toolkit.plot_match(image, template, location.center_in_pixels)

Locate a template in the image using Histogram of gradients template matcher and inspect the result:


    from autoscript_toolkit.template_matchers import *
    
    # Acquire one full-frame image
    settings = GrabFrameSettings(resolution="1536x1024", dwell_time=1e-6)
    image = microscope.imaging.grab_frame(settings)
    
    # Acquire one reduced area image to serve as a template
    settings.reduced_area = Rectangle(0.05, 0.05, 0.10, 0.10)
    template = microscope.imaging.grab_frame(settings)
    
    # Locate the template in the original image
    hog_matcher = HogMatcher(microscope)
    location = vision_toolkit.locate_feature(image, template, hog_matcher)
    ui_toolkit.plot_match(image, template, location.center_in_pixels)
