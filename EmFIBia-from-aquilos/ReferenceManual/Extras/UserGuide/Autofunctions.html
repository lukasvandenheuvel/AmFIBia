User guide: Auto functions

This chapter demonstrates how to execute auto functions, which are automated routines designed to optimize the microscope optics and detector parameters for the highest quality of acquired images.

Before proceeding with the code snippets below, initialize AutoScript with the following code. Set the server address according to your environment. 


    from autoscript_sdb_microscope_client import SdbMicroscopeClient
    from autoscript_sdb_microscope_client.enumerations import *
    from autoscript_sdb_microscope_client.structures import *
    
    microscope = SdbMicroscopeClient()
    microscope.connect("localhost")

Setup imaging environment:


    microscope.imaging.set_active_view(1)
    microscope.imaging.set_active_device(ImagingDevice.ELECTRON_BEAM)

Run the auto contrast/brightness routine in the active view using the default algorithm method:


    microscope.auto_functions.run_auto_cb()

Run the auto contrast/brightness routine using the specialized MaxContrast method:


    settings = RunAutoCbSettings(method="MaxContrast")
    microscope.auto_functions.run_auto_cb(settings)

Run the auto contrast/brightness routine using the specialized MaxContrast method and provide specific settings:


    settings = RunAutoCbSettings(method="MaxContrast", resolution="192x128", calibrate_detector=True)
    microscope.auto_functions.run_auto_cb(settings)

Run the auto stigmator routine in the OngEtAl variant:


    microscope.auto_functions.run_auto_stigmator(RunAutoStigmatorSettings(method="OngEtAl"))

Run the auto stigmator centering routine using the Volumescope method:


    microscope.auto_functions.run_auto_stigmator_centering(RunAutoStigmatorCenteringSettings(method="Volumescope"))

Other auto functions can be performed in a similar manner. For more information about all available auto functions and their settings, see the reference manual, section auto_functions.
