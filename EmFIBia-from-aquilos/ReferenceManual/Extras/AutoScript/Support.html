How to report a bug

To ensure efficient bug resolution, please follow these steps:

  1. Open the AutoScript Diagnostic Report application from the Start menu and use it to create a zip file containing diagnostic information. If you are working in a two-computer setup (the remote scripting configuration), it is beneficial to create the diagnostic report on both computers.
  2. During the zip creation, you will be prompted to describe the problem you are experiencing. Include details such as error messages, unexpected behavior of the microscope, and the frequency of the issue.
  3. Include relevant parts of the script you are using to better illustrate the problem. Add screenshots if they help explain the issue visually.
  4. Attach the files to an e-mail and send it to the support contact e-mail listed below.

Support contacts
