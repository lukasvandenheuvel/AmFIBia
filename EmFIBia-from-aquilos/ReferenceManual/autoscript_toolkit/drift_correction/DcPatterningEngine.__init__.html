DcPatterningEngine.__init__

Initializes a new instance of DcPatterningEngine.

Syntax

__init__(microscope, template_matcher=None)

Parameters

microscope SdbMicroscopeClient The initialized (connected) instance of the SdbMicroscopeClient class.
template_matcher TemplateMatcher The template matcher to be used for locating the fiducial. If not provided, the engine will default to using the OpenCV template matcher.

Examples


        import autoscript_toolkit.drift_correction as dc_toolkit
        
        # Create the engine with the default (Open-CV based) template matching algorithm
        dcp_engine = dc_toolkit.DcPatterningEngine(microscope)
        
        # Create another engine, now with the histogram-of-gradients template matching algorithm
        template_matcher = HogMatcher(microscope)
        dcp_engine = dc_toolkit.DcPatterningEngine(microscope, template_matcher=template_matcher)
