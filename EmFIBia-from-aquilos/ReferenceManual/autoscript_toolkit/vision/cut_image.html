cut_image

Cuts out a portion of an image according to the specified target area definition.

Syntax


        cut_image(image, top_left, bottom_right, center, size, relative_center, relative_size)

Parameters

image AdornedImage | numpy.ndarray The image from which a part will be cut out.
top_left Point | list | tuple The top-left coordinate of the area to be cut out, specified in pixels.
bottom_right Point | list | tuple The bottom-right coordinate of the area to be cut out, in pixels.
center Point | list | tuple The center of the cut area, in pixels.
size Point | list | tuple The size of the cut area, in pixels.
relative_center Point | list | tuple The center of the cut area, in relative coordinates in the range [0, 1].
relative_size Point | list | tuple The size of the cut area, in relative coordinates in the range [0, 1].

Return value

AdornedImage | numpy.ndarray The image region extracted from the input image, provided in the same format as the input image.

Remarks

  • Only the following combinations of parameters are valid: (top_left and bottom_right) OR (center and size) OR (relative_center and relative_size).
  • The function accepts both AdornedImage and numpy.ndarray for the image parameter. The resulting cut-out image is provided in the same form.

Examples


        import autoscript_toolkit.vision as vision_toolkit
    
        # Acquire a FIB image
        microscope.imaging.set_active_device(ImagingDevice.ION_BEAM)
        fib_image = microscope.imaging.grab_frame(GrabFrameSettings(resolution=ScanningResolution.PRESET_1536X1024))
        
        # Extract a fiducial image as a rectangular region, providing coordinates of the top-left and bottom-right corners
        fiducial_image = vision_toolkit.cut_image(fib_image, top_left=[100, 100], bottom_right=[200, 200])
        
        # Perform the same operation by specifying the center point and size of the region
        fiducial_image = vision_toolkit.cut_image(fib_image, center=[150, 150], size=[100, 100])
