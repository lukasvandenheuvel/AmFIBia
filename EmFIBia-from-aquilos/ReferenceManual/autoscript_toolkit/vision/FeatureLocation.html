FeatureLocation

Contains information about a feature located within an image. The function locate_feature() returns this type.

Methods

print_all_information() Prints all available information in a human-readable form.

Properties

confidence The level of confidence with which the feature was located, expressed as a value in the range of [0, 1], corresponding to 0% - 100%.
center_in_pixels Coordinates of the center of the feature in the original image, in pixels.
center_in_meters Coordinates of the center of the feature in the original image, in meters. Additional details regarding the coordinate system are provided in the notes below.
shift_in_pixels Offset of the feature from its original position, expressed in pixels.
shift_in_meters Offset of the feature from its original position, expressed in meters.

Remarks

  • The coordinate system utilized for properties that reference meters (center_in_meters, shift_in_meters), mirrors the coordinate system employed for patterns. In this system, the origin is located at [0, 0], positioned in the center of the field of view. X-axis coordinates progress from left to right, while Y-axis coordinates increase from bottom to top.
  • The values of shift_in_pixels and shift_in_meters are only valid when the original position of the feature is provided as a parameter to the locate_feature() function.

Examples


        import autoscript_toolkit.vision as vision_toolkit
        import autoscript_toolkit.user_interface as ui_toolkit

        # Take a full frame image and prepare a template image
        settings = GrabFrameSettings(resolution="1536x1024", dwell_time=1e-6)
        searched_image = microscope.imaging.grab_frame(settings)
        template_image = vision_toolkit.cut_image(searched_image, relative_center=[0.40, 0.40], relative_size=[0.20, 0.20])

        # Locate the feature in the original image using the default (OpenCV based) template matcher
        location = vision_toolkit.locate_feature(searched_image, template_image)

        # Inspect the result
        location.print_all_information()
        ui_toolkit.plot_match(searched_image, template_image, location.center_in_pixels)
