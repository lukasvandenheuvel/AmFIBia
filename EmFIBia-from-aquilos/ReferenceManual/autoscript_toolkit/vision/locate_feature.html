locate_feature

Finds the position of the specified template image within another image.

Syntax


        locate_feature(searched_image, template_image, template_matcher=None, original_feature_center=None)

Parameters

searched_image AdornedImage The image in which the feature is to be located.
template_image AdornedImage The template of the feature to be located.
template_matcher TemplateMatcher An instance of the TemplateMatcher class, to be used for matching the template against the searched image. If not provided, the default template matcher OpencvTemplateMatcher is used.
original_feature_center Point | list | tuple The center of the original feature location in pixels. If provided, the resulting FeatureLocation structure will contain the feature shift data. When not provided, only the current feature position is returned.

Return value

FeatureLocation A data class containing details of the feature location.

Remarks

When using the OpenCV-based template matching algorithm, the result object does not include a confidence factor; it will be set to None. When configured with HOG or Cognex algorithms, the confidence level is provided.

Examples


        import autoscript_toolkit.vision as vision_toolkit
        from autoscript_toolkit.template_matchers import HogMatcher
        
        # Take a full-frame image
        settings = GrabFrameSettings(resolution="1536x1024", dwell_time=1e-6)
        searched_image = microscope.imaging.grab_frame(settings)
        
        # Take a reduced-area image to serve as a template
        settings.reduced_area = Rectangle(0.05, 0.05, 0.10, 0.10)
        template_image = microscope.imaging.grab_frame(settings)
        
        # Locate the template in the full-frame image using the OpenCV matcher
        feature_location = vision_toolkit.locate_feature(searched_image, template_image)
        feature_location.print_all_information()
        
        # Locate the template again, now with the histogram-of-gradients matcher
        hog_matcher = HogMatcher(microscope)
        feature_location = vision_toolkit.locate_feature(searched_image, template_image, hog_matcher)
        feature_location.print_all_information()
