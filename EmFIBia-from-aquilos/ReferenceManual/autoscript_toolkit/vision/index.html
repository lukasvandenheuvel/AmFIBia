Vision

This module provides a range of functions tailored for computer vision tasks specific to electron microscopy. These functions are designed to work directly with the AutoScript representation of microscope images (the AdornedImage classes), eliminating the need for conversions in client code.

Image processing routines can be executed either within Python code on the client side of AutoScript, as seen in the locate_feature() function using the OpenCV variant of the pattern matcher. Alternatively, some processing algorithms are implemented on the AutoScript server side, requiring data transmission for processing. This applies to the locate_feature() function configured to use the histogram-of-gradients pattern matcher implementation, as well as the blend_images() function.

Classes

FeatureLocation Contains information about a feature located within an image.

Functions

blend_images() Combines two images together, optionally resizing them to a target size.
connect() Assigns a SdbMicroscopeClient object to the Vision Toolkit, enabling functions reliant on server-side image processing routines.
cut_image() Cuts out a portion of an image according to the specified target area definition.
locate_feature() Finds the position of the specified template image in an image.
