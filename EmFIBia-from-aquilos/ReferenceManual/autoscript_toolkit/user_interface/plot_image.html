plot_image

Opens a pop-up window displaying a plot of the provided image, given as an AdornedImage instance. The image is rendered using a grayscale palette.

Syntax

        plot_image(image, label=None)

Parameters

image AdornedImage The image to be displayed.
label str The title of the window displaying the image plot. If not specified, the title is derived from the detector name in the image metadata.

Examples


        import autoscript_toolkit.user_interface as ui_toolkit
        
        # Acquire a partial image
        settings = GrabFrameSettings(reduced_area=[0.1, 0.2, 0.2, 0.2])
        image = microscope.imaging.grab_frame(settings)

        # Display it in a popup window
        ui_toolkit.plot_image(image)
