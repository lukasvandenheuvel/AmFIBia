User interface

This module provides a variety of functions designed for displaying electron microscopy images and facilitating user interaction.
These functions are designed to work directly with the AutoScript representation of microscope data (e.g. AdornedImage and AdornedSpectrum classes), eliminating the need for conversions in client code.

Functions

plot_image() Opens a pop-up window displaying a plot of the provided image, given as an AdornedImage instance. The image is rendered using a grayscale palette.
plot_images() Opens a pop-up window displaying a combined plot of the provided images, given as AdornedImage instances. The images are rendered using a grayscale palette.
plot_match() Displays a searched image and a feature located in it in a popup window.
plot_spectra() Opens a pop-up window displaying a plot where the provided spectra, given as AdornedSpectrum instances, are overlaid in a single figure.
plot_spectrum() Opens a pop-up window displaying a plot of the provided EDS spectrum, given as an AdornedSpectrum instance.
