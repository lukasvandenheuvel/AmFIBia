plot_match

Displays a popup window with the result of a pattern matching task, showing the template image alongside the searched image with the position of the template marked. The function often serves as a troubleshooting tool for debugging.

Syntax


        plot_match(searched_image, template_image, match_center)

Parameters

searched_image AdornedImage The image in which the feature was located.
template_image AdornedImage The template of the feature that was located.
match_center Point | list | tuple The center of the feature, specified in pixels.

Examples


        import autoscript_toolkit.user_interface as ui_toolkit
        import autoscript_toolkit.vision as vision_toolkit
        from autoscript_toolkit.template_matchers import HogMatcher

        # Take a full-frame image
        settings = GrabFrameSettings(resolution="1536x1024", dwell_time=1e-6)
        searched_image = microscope.imaging.grab_frame(settings)
        
        # Take a reduced-area image to serve as a template
        settings.reduced_area = [0.655, 0.4, 0.16, 0.26]
        template_image = microscope.imaging.grab_frame(settings)
        
        # Locate the template with the histogram-of-gradients matcher
        hog_matcher = HogMatcher(microscope)
        feature_location = vision_toolkit.locate_feature(searched_image, template_image, hog_matcher)
        
        # Display the searched image with the template position marked
        ui_toolkit.plot_match(searched_image, template_image, feature_location.center_in_pixels)
