plot_images

Opens a pop-up window displaying a combined plot of the provided images, given as AdornedImage instances. The images are rendered using a grayscale palette.

Syntax


        plot_images(images, labels=None)

Parameters

images List[AdornedImage] The list of images to be displayed in the combined plot. The minimum number of images is 1, the maximum is 9.
labels List[str] A list of labels for annotating each image. If unspecified, the labels are generated from the detector names found in image metadata.

Examples


        import autoscript_toolkit.user_interface as ui_toolkit
        
        # Configure views with different detectors for multi-detector image acquisition
        # ...
        
        # Acquire one image from each view, up to a total of 4 images
        images = microscope.imaging.grab_multiple_frames()
        
        # Display the images in a combined plot
        ui_toolkit.plot_images(images)
