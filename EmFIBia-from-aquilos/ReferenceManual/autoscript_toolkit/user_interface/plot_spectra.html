plot_spectra

Opens a pop-up window displaying a plot where the provided spectra, given as AdornedSpectrum instances, are overlaid in a single figure.

Syntax


        plot_spectra(spectra, labels=None)

Parameters

spectra List[AdornedSpectrum] The list of spectra to be displayed in the combined plot. The minimum number of spectra is 1, the maximum is 10.
labels List[str] A list of labels for annotating each spectrum. If left unspecified, it will be automatically generated based on the experiment name in spectrum metadata.

Examples


        import autoscript_toolkit.user_interface as ui_toolkit
        
        # Acquire multiple spectra
        region1 = Region(type=RegionType.RECTANGLE, rectangle=[0, 0, 0.5, 1], coordinate_system=RegionCoordinateSystem.REDUCED_AREA)
        spectrum1 = microscope.analysis.eds.acquire_spectrum(region1, 5)

        region2 = Region(type=RegionType.RECTANGLE, rectangle=[0.5, 0, 0.5, 1], coordinate_system=RegionCoordinateSystem.REDUCED_AREA)
        spectrum2 = microscope.analysis.eds.acquire_spectrum(region2, 5)

        spectra = [spectrum1, spectrum2]
        
        # Display the spectra in a combined plot
        ui_toolkit.plot_spectra(spectra)
