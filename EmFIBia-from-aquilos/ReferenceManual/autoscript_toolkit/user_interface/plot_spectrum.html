plot_spectrum

Opens a pop-up window displaying a plot of the provided EDS spectrum, given as an AdornedSpectrum instance.

Syntax


        plot_sectrum(spectrum, label=None)

Parameters

spectrum AdornedSpectrum The spectrum to display.
label str The title of the window displaying the spectrum. If left unspecified, it will be automatically generated based on the experiment name in spectrum metadata.

Examples


	import autoscript_toolkit.user_interface as ui_toolkit
	
    # Read an EDS spectrum collected from previous scanning activity
    spectrum = microscope.analysis.eds.get_spectrum()
    
    # Display the spectrum in a pop-up window
    ui_toolkit.plot_spectrum(spectrum)
