AutoScript toolkit

The AutoScript toolkit is a Python package designed to provide robust data processing capabilities in specific areas of electron microscopy. This package is an integral part of the broader AutoScript software suite. For comprehensive information about AutoScript, see the other sections of this reference manual, especially the Microscope API package.

This toolkit consists of four different modules. Image-template matchers can be found in the module Template matchers. Help for computer vision tasks is provided through Vision package with a combination of User interface package. Another package, Drift correction, applies computer vision to a specific area of the sample drift correction.
