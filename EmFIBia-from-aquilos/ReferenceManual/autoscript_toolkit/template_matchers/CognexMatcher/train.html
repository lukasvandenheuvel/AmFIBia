CognexMatcher.train

Trains the Cognex PatMax algorithm on a template image, allowing for detailed training settings to be specified.

Syntax

train(template: AdornedImage, settings: CognexMatcherSettings = None)

Parameters

template AdornedImage Template image to be matched within the searched image in further runs (calls to match()).
settings CognexMatcherSettings Settings for the PatMax algorithm training.

Remarks

The structure CognexMatcherSettings and enumerations CognexTrainAlgorithm, CognexTrainMode, CognexRunAlgorithm, CognexRunMode, and CognexSearchRegionMode are declared but not yet documented in the current version of AutoScript.

Examples

See CognexMatcher.
