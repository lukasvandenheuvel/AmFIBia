CognexMatcher

This class represents an image template matcher based on the Cognex PatMax algorithm.

Properties

is_installed [Read only] Returns true if the Cognex VisionPro software is installed on the microscope computer.
is_licensed [Read only] Returns true if the Cognex VisionPro software is licensed on the microscope computer.

Methods

__init__() Creates a new CognexMatcher.
match() Matches the given template image within the given image.
train() Trains the Cognex PatMax algorithm on a template image, allowing for detailed training settings to be specified.

Remarks

  • PatMax is a robust feature location algorithm from Cognex's VisionPro machine vision library. This algorithm is designed to accurately locate features even if they have undergone transformations such as rotation or resizing.
  • VisionPro software is not installed with the base AutoScript product. Its lisense is not included in any AutoScript product license variant.
  • VisionPro must be installed manually on the microscope computer alongside the AutoScript server component. The required VisionPro version is 9.5 or newer, 64-bit edition.
  • For more information about VisionPro, visit the vendor's site: https://www.cognex.com/.

Examples


        import autoscript_toolkit.vision as vision_toolkit
        import autoscript_toolkit.user_interface as ui_toolkit
        from autoscript_toolkit.template_matchers import *
        from autoscript_toolkit_cognex_extension.cognex_template_matcher import *

        # Acquire one full-frame image in the active view
        settings = GrabFrameSettings(resolution="1536x1024", dwell_time=1e-6)
        image = microscope.imaging.grab_frame(settings)
        
        # Acquire one reduced area image to serve as a template
        settings.reduced_area = Rectangle(0.05, 0.05, 0.10, 0.10)
        template = microscope.imaging.grab_frame(settings)
        
        # Locate the template in the original image with default PatMax training strategy
        cognex_matcher = CognexMatcher(microscope)
        location = vision_toolkit.locate_feature(image, template, cognex_matcher)
        ui_toolkit.plot_match(image, template, location.center_in_pixels)
        
        # Repeat the search, but with a different training algorithm
        patmax_settings = CognexMatcherSettings(train_algorithm=CognexTrainAlgorithm.PAT_MAX_AND_PAT_QUICK)
        cognex_matcher.train(template, patmax_settings)
        location = vision_toolkit.locate_feature(image, template, cognex_matcher)
        vision_toolkit.plot_match(image, template, location.center_in_pixels)
