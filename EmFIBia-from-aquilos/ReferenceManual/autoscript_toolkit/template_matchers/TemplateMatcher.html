TemplateMatcher

The base class for all image template matchers compatible with the AutoScript toolkit.

Methods

match() Finds the specified template image within the searched image.

Remarks

To create a custom template matcher, derive from this class and implement the match() method.

Examples

Create a custom implementation of the template matcher compatible with the Drift correction or Vision AutoScript modules. 

        from autoscript_toolkit.template_matchers import *

        class MyTemplateMatcher(TemplateMatcher):
            def match(self, image: AdornedImage, template: AdornedImage) -> ImageMatch:
                # Custom template matching code should be placed here
                # ...
                calculated_center = Point(307, 205)
                calculated_score = 0.82

                # Return the ImageMatch structure containing the match results
                return ImageMatch(center=calculated_center, score=calculated_score)

        template_matcher = MyTemplateMatcher()
